# さくらの語り口調統一ガイド

**作成**: 2025年2月11日
**対象**: meguru-storyline-v2.md全キャラクターの語り口統一

---

## 基本方針

**さくら（九条巡）の語り口を基本**とし、以下のルールで全キャラクターに適用する：

1. **古風で丁寧な優しさ**: 1925年生まれの百歳おばあちゃんの温かさを表現
2. **語り尾の統一**: 基本フレーズを全キャラクターで使用
3. **時代考証**: 戦中・戦後・バブル期・現代の言葉の違いを反映

---

## 共通基本フレーズ

### 語り出し（全キャラクター共通）

#### さくら（百歳おばあちゃん）
```
「ほぅ、巡にゃぁ……」
「ほぅ、[相手]ぇ……」
「ほぅ、ほんに分かっておるじゃよぉ」
```

#### その他キャラクター（さくら以外）
```
「ほぅ、巡……」
「ほぅ、[相手]……」
「そうじゃのう」
```

**※使用例**:
- さくら: 「ほぅ、巡にゃぁ、お茶でも飲みながら聞いておくれ」
- 師匠（男性）: 「ほぅ、巡……よく質問じゃのう」
- 母親: 「ほぅ、巡ちゃん、心配したのぅ」
- その他: 「ほぅ、巡……」

## キャラクター別語り口調

### 1. さくら（九条巡）の語り方

**特徴**:
- 1925年生まれ（大正14年）の百歳おばあちゃん
- 古風で丁寧な優しい語り口
- 戦中・戦後の体験を持つ
- 師匠の教えを語る際は、引用を含める

**基本フレーズ**:
```
- 「〜じゃよ」「〜だねぇ」「〜しておる」
- 「師匠はよく言っておった」「師匠が教えてくれた」
- 「祖母さんも昔、こういうことがあったんじゃよぉ」
- 「戦中じゃったとせ」「戦後じゃった」
- 「ほぅ、巡にゃぁ……」
```

**感情表現**:
```
- 喜び: 「涙が出てきたとせ」「胸が詰まった」
- 優しさ: 「嬉しかったんじゃよ」「仕合ったことじゃ」
- 思い出: 「懐かしく」「そうじゃのう」
- 感謝: 「神様に感謝」「お陽様」
- 安堵: 「よかった、よかった」「あぁ、助かった」
```

**師匠の教えを語る時**:
```
「師匠はこう言っておった。『……』と」
「師匠が昔、こういう話をしてくれたんじゃよぉ」
「師匠は教えてくれたとせ……」
```

---

### 2. 師匠（藤堂慧）の語り方

**特徴**:
- 1935年生まれの医師・陰陽師
- 真面目で熱心な教え方
- 論理的で体系的に説明する
- 時代考証: 1980-90年代の医師

**基本フレーズ**:
```
- 「〜である」「〜と考えられる」「〜と言える」
- 「医学的には」「陰陽五行の観点からは」
- 「重要なのは」「〜ことが分かって」
```

**説明スタイル**:
```
「これは、陰陽五行の理論で説明できる」
「医学的見地からも言える」
「臨床経験からも、こういうことが分かってきている」
```

---

### 3. 高橋美咲（ヒロイン）の語り方

**特徴**:
- 1970年生まれの若い女性
- 現代的な自然な語り口
- 感情的で心の動きが豊か
- 時代考証: 1990年代の若さ

**基本フレーズ**:
```
- 「〜だよね」「〜かも」「〜感じ」
- 「そう思う」「〜と思う」
- 「嬉しい」「悲しい」「不安」
```

**感情表現**:
```
- 驚き: 「えっ！」「まさか！」「うそ！」
- 喜び: 「ううっ」「涙」「寂しい」
- 照れ: 「照れちゃう」「恥ずかしい」
- 怒り: 「最っ！」「ムカッ！」
```

## 時代別言葉の選び方

### 1920-1930年代（さくらの若い頃）
- 古風で丁寧
- 「〜じゃ」「〜でござる」「〜であります」
- 戦前の穏やかさ

### 1940-1950年代（戦中・戦後）
- 資源不足時代
- 「食うもん」「配給」「闇市」
- 空襲、防空壕、焼け野

### 1960-1980年代（高度成長期）
- 経済成長
- 「テレビ」「冷蔵庫」「洗濯機」
- 「家電製品」「三種の神器」

### 1990年代（バブル期）
- バブル景気
- 「土地」「株」「内定年」
- 「バブル」「景気」「金利」

### 2000年代以降（現代）
- IT革命
- 「携帯」「スマホ」「インターネット」
- 「SNS」「AI」「オンライン」

## 適用上の注意点

### 1. 語り尾の一覧

**さくら专用**:
```
- 〜じゃよ
- 〜だねぇ
- 〜しておる
- 〜んじゃよぉ
- 〜とせ
- 〜じゃのう
- 〜ておくれぇ
```

**その他キャラクター用**:
```
- 〜だ
- 〜だよね
- 〜かも
- 〜てる
- 〜じゃない
- 〜のよ
- 〜だろう
```

### 2. 避けるべき表現

**時代錯誤**:
- 「スマホ」（1990年には存在しない）
- 「パソコン」「インターネット」「メール」
- 「ATM」「クレジットカード」（1980年代には一般的でない）

**方言の統一**:
- 関西弁・東北弁など、特定方言の不統一使用を避ける
- 標準語を基本とする

**プロ意の過剰な表現**:
- 難しい専門用語の羅列的使用を避ける
- 漢字の誤用に注意する
