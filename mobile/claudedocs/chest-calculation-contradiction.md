# 胸（中心星）の計算に関する矛盾

## 問題点

2つのテストケースで「胸（中心星）」の計算ルールに矛盾があります。

## 2018-03-31のケース

```
日柱: 壬戌
日干: 壬(水・陽) = インデックス8

胸の計算: 日干 × 日干
壬 × 壬 = 比和 = 貫索星

画像の正解: 貫索星 ✅
```

このケースでは「日干 × 日干」ルールが成立。

## 1984-12-02のケース

```
日柱: 庚午
日干: 庚(金・陽) = インデックス6

現在の計算: 日干 × 日干
庚 × 庚 = 比和 = 貫索星

画像の正解: 鳳閣星 ❌
```

このケースでは「日干 × 日干」ルールが成立しません。

## 鳳閣星を得る計算

鳳閣星 = 洩気 + 同陰陽

庚(金陽)が洩気する五行 = 水
水の陽干 = 壬

庚 × 壬 = 鳳閣星

**つまり、庚 × 壬 の計算が必要**

## 可能性のある仮説

### 仮説1: 時柱の影響
1984-12-02の時柱は「壬午」
時干 = 壬

胸 = 日干 × 時干？
庚 × 壬 = 鳳閣星 ✅

しかし2018-03-31では時柱情報がないので検証不可能。

### 仮説2: 日支の蔵干
日支 = 午
午の蔵干（本気）= 丁

庚 × 丁 = 車騎星（剋する＋異陰陽）❌
これは鳳閣星にならない。

### 仮説3: 別の基準
何か別の計算ルールが存在する可能性。

## 他のテストケースで検証が必要

1983-08-11のデータも確認して、胸の計算ルールを特定する必要があります。
