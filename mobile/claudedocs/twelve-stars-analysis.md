# 十二大従星の位相計算分析

## 2018-03-31のデータから逆算

### 基本情報
```
日干: 壬 (陽・水=4)
年支: 戌 (土=2)
月支: 卯 (木=0)
日支: 戌 (土=2)
```

### 正解データ
```
左肩（年支=戌）: 天南星 (10点)
右足（月支=卯）: 天南星 (10点)
左足（日支=戌）: 天極星 (2点)
```

## 問題点

**同じ支（戌）なのに異なる星**:
- 年支=戌 → 天南星(10)
- 日支=戌 → 天極星(2)

これは以下のいずれかの可能性：

### 仮説1: 位置による補正がある
```typescript
左肩（年支）: 基本位相 + 0
右足（月支）: 基本位相 + X
左足（日支）: 基本位相 + Y
```

### 仮説2: 年・月・日支で異なる蔵干を使う
```
年支=戌: 本気=戊 を使う
日支=戌: 中気=辛 を使う
```

### 仮説3: 支そのものの五行で計算
```
戌の五行 = 土(2)
壬の五行 = 水(4)
(2 - 4 + 5) % 5 = 3
```

## 五行差と期待される星の対応

### 卯（木=0）から天南星(10)を得る
```
日干: 壬 (水=4)
月支: 卯 (木=0)
五行差: (0 - 4 + 5) % 5 = 1
```
**五行差=1 で 天南星(10)**

### 戌（土=2）から天南星(10)を得る
```
日干: 壬 (水=4)
年支: 戌 (土=2)
五行差: (2 - 4 + 5) % 5 = 3
```
**五行差=3 で 天南星(10)**

### 戌（土=2）から天極星(2)を得る
```
日干: 壬 (水=4)
日支: 戌 (土=2)
五行差: (2 - 4 + 5) % 5 = 3
```
**五行差=3 で 天極星(2)**

## 矛盾

同じ五行差(3)で異なる星が出ている。
→ **位置による補正が必要**
