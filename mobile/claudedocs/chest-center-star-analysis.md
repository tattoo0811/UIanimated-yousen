# 胸（中心星）計算の詳細分析

## 3つのテストケースの比較

### ケース1: 2018-03-31
```
日柱: 壬戌
日干: 壬(水・陽) インデックス8
時柱: なし

現在の計算: 壬 × 壬 = 比和 = 貫索星
画像の正解: 貫索星 ✅

このケースでは「日干×日干」が成立
```

### ケース2: 1984-12-02
```
日柱: 庚午
日干: 庚(金・陽) インデックス6
時柱: 壬午
時干: 壬

現在の計算: 庚 × 庚 = 比和 = 貫索星
画像の正解: 鳳閣星 ❌

鳳閣星を得るには:
庚 × 壬 = 洩気 + 同陰陽 = 鳳閣星
→ 時干を使っている？
```

### ケース3: 1983-08-11
```
日柱: 辛未
日干: 辛(金・陰) インデックス7
時柱: 甲午
時干: 甲

現在の計算: 辛 × 辛 = 比和 = 貫索星
画像の正解: 玉堂星 ❌

玉堂星を得るには:
辛 × 己 = 生じられる + 同陰陽 = 玉堂星

日支 = 未
未の蔵干(本気) = 己
辛 × 己 = 玉堂星 ✅
→ 日支の蔵干を使っている？
```

## 仮説

### 仮説1: 時柱がある場合は時干を使う
```
2018-03-31: 時柱なし → 日干×日干 ✅
1984-12-02: 時柱あり(壬午) → 日干×時干(庚×壬) = 鳳閣星 ✅
1983-08-11: 時柱あり(甲午) → 日干×時干(辛×甲) = 牽牛星 ❌
```
1983-08-11で不成立

### 仮説2: 日支の蔵干(本気)を使う
```
2018-03-31: 日支=戌, 蔵干=戊 → 壬×戊 = 石門星 ❌
1984-12-02: 日支=午, 蔵干=丁 → 庚×丁 = 車騎星 ❌
1983-08-11: 日支=未, 蔵干=己 → 辛×己 = 玉堂星 ✅
```
1983-08-11のみ成立

### 仮説3: 条件付き分岐
```
IF 時柱が存在 THEN
  IF 時干との計算が特定条件を満たす THEN
    日干×時干
  ELSE
    日干×日支蔵干
ELSE
  日干×日干
```

## 追加検証が必要な点

1. 1984-12-02で「日干×時干」が成立する理由
2. 1983-08-11で「日干×日支蔵干」が成立する理由
3. 2018-03-31に時柱データがあるかどうか
4. 他のテストケース(2016-10-27)の検証

## 重要な発見

**胸（中心星）の計算ルールは単純な「日干×日干」ではなく、より複雑な条件分岐がある可能性が高い**
