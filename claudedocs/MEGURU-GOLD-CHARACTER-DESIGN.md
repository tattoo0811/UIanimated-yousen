## 1. 辛金（金陰）としての特性

### 基本性質

```
辛金とは：
- 宝石・装飾品・精巧な工芸品を象徴
- 壬金（大剛・鉄）とは異なり、「磨かれてこそ輝く」性質
- 美しさ・完成度・繊細さ・プライドを持つ
- 頑固だが、理解されれば深い忠誠を誓う
```

### 五行の循環における位置

```
土 → 金 → 水 → 木 → 火 → 土
      ↑
    巡（辛金）

相生関係:
- 土から生まれる（土→金）：未（土）が辛（金）を支える
- 水を生む（金→水）：辛金が持つ知性・適応力

相克関係:
- 火に剋される（火剋金）：試練・変容を通じて成長
- 木を剋する（金剋木）：美咲（木）との関係で重要
```

## 3. 裏切りによる「金の傷」

### 裏切りが金（辛）に与えるダメージ

| 金の特性 | 裏切りによる傷 | その後の影響 |
|---------|---------------|-------------|
| **プライド** | 「信頼していた人に裏切られる」 | 自己不信・他者不信 |
| **輝き** | 「宝石が泥にまみれる」 | 自分の価値を疑う |
| **完成** | 「不完全な決別」 | 閉ざこもる・暗い |
| **頑固さ** | 「許せない」という執着 | うつ状態の長期化 |

### 裏切りのシーン（金としての巡の反応）

```

「法的にも君は負ける。堅い契約書もある」

巡の反応（金としての特性）:
1. **沈黙**（辛金は感情的に騒がない）
   - 黙って契約書を読む
   - 内容の正確さを分析（論理的）

2. **プライドの傷つき**
   - 「法的に負ける」という言葉が刺さる
   - 金の「尊厳」を傷つけられた感覚

3. **反論できない弱さ**
   - 確かに資金は慧が多く出していた
   - 技術も半分は慧の貢献
   - 「正論」に完膚なきまでに論破される

4. **宝石が泥に**
   - 巡の「医療への志し」が「ビジネスの論理」に踏みにじられる
   - 「完璧な友情」が「裏切り」という不完全な現実に

巡の最後の言葉:
「……わかった。君の言う通りだ」

（静かに退室する）

→ 辛金の「辛抱強さ」・「脆さ」が同時に表現
```
