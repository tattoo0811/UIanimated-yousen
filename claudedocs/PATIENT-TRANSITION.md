## 1. 標準的な患者の状態遷移

```mermaid
stateDiagram-v2
    [*] --> 悩み抱く: 日常生活での悩み
    悩み抱く --> 来院: 運命診断室へ
    来院 --> カルテ記入: 生年月日・経歴
    カルテ記入 --> 診断: 巡が命式を診断
    診断 --> 説明: 陰陽五行からの説明
    説明 --> 処方箋: 陰陽バランスの処方箋
    処方箋 --> 実行: 患者が実行
    実行 --> 変化: 少しずつ変化
    変化 --> 解決: 問題が解決
    解決 --> フォローアップ: 数ヶ月後に再診
    フォローアップ --> [*]: 新しい段階へ

    note right of 悩み抱く
        仕事・恋愛・家族・健康
        「どうしたらいいか分からない」
        限界状態
    end note

    note right of 来院
        スマホで発見
        SNSで知る
        友人から紹介
    end note

    note right of 診断
        命式の作成
        五行バランスの確認
        十大主星・天中殺の分析
    end note

    note right of 説明
        「あなたの命式は...」
        陰陽五行からの説明
        「悪くない」の強調
    end note

    note right of 処方箋
        「処方：...」
        具体的な行動提案
        バランス修正の方法
    end note

    note right of 実行
        週に一度の実践
        少しずつの変化
        継続の重要性
    end note

    note right of 変化
        「少しだけ気持ちが軽い」
        周囲の反応が変わる
        自分の変化に気づく
    end note

    note right of 解決
        問題が解決
        新しい視点の獲得
        「巡との出会いが人生を変えた」
    end note

    note right of フォローアップ
        数ヶ月後に再診
        その後の変化を報告
        新しい段階での悩み
    end note
```
