## 第1話: 土の下の春

| 項目 | 内容 |
|------|------|
| **患者** | 高橋美咲（27歳・乙卯）「辞めるべきか悩む広告代理店」 |
| **トリガー** | 美咲が「数字で評価される仕事に、自分の価値を見出せない」と吐露する瞬間 |
| **回想内容** | 2008年春、医学部4年。巡は外科医を目指すが、父・篤人から「精神科も考えてみろ」と言われる。「患者の心を見るのも医者の仕事だ」。巡は反発する──「僕は病変を切り取る外科医になりたい」。しかし臨床実習で、あるうつ状態の患者と向き合う。患者は「何も悪くないのに、何もできない」と呟く。巡は理解できない。ただ「頑張ってください」としか言えなかった。その無力感が、今の美咲に重なる。 |
| **情報の断片** | ・巡は元々外科医志望だった<br>・父は開業医<br>・学生時代に「理解できない心の痛み」に遭遇 |
| **感情トーン** | 自分の無力さを再認識する痛み。美咲への共感は、過去の自分への共感でもある |

| **情報の断片** | ・「栞」という名の同僚がいた<br>・患者の自殺未遂に直面したトラウマ<br>・完璧主義がうつの原因の一つ |
