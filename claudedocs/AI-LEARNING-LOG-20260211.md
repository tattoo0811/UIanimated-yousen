### 間違い2: 大運順逆ルールの誤解

**間違いった理解**:
```
❌ 誤った理解:
- さくら（乙年生まれ、女性）→ 逆行になると誤解
- 美咲（己年生まれ、女性）→ 逆行になると誤解
```

**正しいルール**:
```typescript
// 大運順逆の正しい判定ロジック
const isMale = gender === 'male';
const isYGanYang = GAN.indexOf(yGan) % 2 === 0; // 0,2,4,6,8 = 陽
const isForward = (isYGanYang && isMale) || (!isYGanYang && !isMale);

// ルール:
// 男性 × 陽年（甲・丙・戊・庚・壬）   → 順行
// 男性 × 陰年（乙・丁・己・辛・癸）   → 逆行
// 女性 × 陰年（乙・丁・己・辛・癸）   → 順行 ⬅️
// 女性 × 陽年（甲・丙・戊・庚・壬）   → 逆行
```

**正しい検証結果**:
| キャラクター | 年干 | 性別 | 陰陽 | 正しい大運 |
|------------|------|------|------|-----------|
| さくら | 乙（陰） | 女 | 陰陰 | **順行** |
| 美咲 | 己（陰） | 女 | 陰陰 | **順行** |

**学び**:
- 「女性＝逆行」という単純なルールではない
- 性別と年干の陰陽の組み合わせで判定する
- 常にルールをコードで確認すべき

## 🔍 陽占は大運順逆に依存しない

**重要な事実**:
- 陽占（十二大従星・十大主星）は**生年月日のみ**で決まる
- 大運の順行・逆行には**影響されない**
- したがって、大運計算の訂正があっても陽占結果は変更不要

**理由**:
- 陽占は「日干」と各干支の関係で決まる
- 大運は「いつどの時期にどの運気を受けるか」を決めるだけ
- それぞれ独立した計算
