**立運計算ロジック**:
```typescript
// 順行の場合
// 誕生日から「次の節入り日」までの日数 ÷ 3 = 立運（切り上げ）
// - 節入り後生まれ: 当月末日までの日数 + 翌月節入り日
// - 節入り前生まれ: 当月の節入り日 - 誕生日

// 逆行の場合
// 誕生日から「前の節入り日」までの日数 ÷ 3 = 立運（切り上げ）
// - 節入り後生まれ: 誕生日 - 当月の節入り日
// - 節入り前生まれ: 前月末日 - 前月節入り日 + 誕生日
```
