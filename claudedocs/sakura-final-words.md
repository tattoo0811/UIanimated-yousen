# さくらの辞世の句候補

**決定済み要素**:
- 九条 さくら：1925年7月30日 - 2024年3月15日（99歳逝去）
- 日干：乙（木）、十二運：天堂星（8点）
- 九条 巡：1990年3月2日、日干：丙（火）、十二運：天将星（12点）
- 関係：木生火（乙木→丙火）、祖母が孫を支える

---

## 和歌（5・7・5・7・7）

### 候補1
```
巡の火は小さとも
消えず燃えよ
乙の木はありて
孫を支えん
```

### 候補2
```
お前はドジの
子だと思へど
その火の種に
宿りてしと
尊き輝きを
見てきたかり
```

### 候補3【推奨】
```
巡の火は
消えず温かく
人の心を
温むりけな
乙木は燃えて
孫を照らす
```

### 候補4
```
九十九年の
旅路果てて
私は行けど
お前の火は
消えずあれよ
```

## 漢詩（自由形式）

### 候補1
```
巡、お前はドジで
いつも私を困らせてくれた
でも、その火は優しくて
人の心を温める火だったの
私はそれをずっと見届けてきた
乙木の私は燃えて
これからは、あなたの火を
誰かの支えにするのよ
迷ったら、その火を頼りに
私の木は、そばに在り続けるから
```

### 候補2
```
巡、あなたの火は小さいかもしれない
でもそれは確かに輝く火
乙の木が燃え尽きることで
火は最も輝くのよ
私は99年、その火を見届けてきた
これからは、お前が誰かの木に
私がしてきたように
支え続ける番なの
お前の火は消えない
私は、そのことを知っているの
```

### 候補3【推奨】
```
巡、お前はドジで
私を困らせばかりだった（笑）
でも、その優しい火は知っているの
お前の火は
人の心を温める一番大事な火
私はずっと、その火を見届けてきた
これからは、自分の足で立ち
誰かの木になりなさい
迷ったら、その温かい火を
お前は持っているのだから
```

## 短い詩

```

```

## 💡 推奨組み合わせ

**和歌形式：候補3**
```
巡の火は
消えず温かく
人の心を
温むりけな
乙木は燃えて
孫を照らす
```
→ 木生火のテーマ明確、リズムも良い

**俳句形式：候補3**
```
お前はドジだ
优しい火よ
人を温かく
```
→ 最も記憶に残る、ユーモアのある中に深いメッセージ

**漢詩形式：候補3**
→ 「人生の知恵」「祖母の愛情」「天職への示唆」を含み、最も物語的に深い

**短い詩：候補4**
→ 最もシンプルで、核心を突く、タイトルや伏線に使いやすい
