---

## 1. プロジェクト概要

### 1.1 目的
「ドクター巡の人生処方箋」120話拡張版の全キャラクターについて、テーマから逆算した適切な生年月日を設計し、算命学的整合性を保証する。

### 1.2 基本方針
1. **テーマ主導**: エピソードテーマ→悩み→適切な算命学特性→生年月日、の順で設計
2. **一貫性**: 既存キャラクターの生年月日はテーマと整合しているか検証
3. **多様性**: 年代、性別、職業、天中殺タイプをバランスよく配置
4. **再登場**: 重要キャラクターは複数エピソードで成長を描く

### 1.3 物語構造
```
導入編(1-5): 開院篇、巡の哲学の確立
基礎編(6-40): 五行、十大主星、十二大従星の深化
葛藤編(41-80): 関係性、天中殺、人生の岐路
統合編(81-96): 社会問題、慧との対決、巡の覚醒
拡張編(97-120): 教育、家族、健康、金銭、住宅、その他
```

## 2. キャラクター設計プロセス

### 2.1 設計フロー
```
ステップ1: エピソードテーマの分析
  ↓
ステップ2: テーマに対応する算命学特性の特定
  - 日干（十日干の性質）
  - 天中殺（人生の転機期）
  - 十大主星（性格傾向）
  - 十二大従星（エネルギーレベル）
  - 五行バランス（エネルギー分布）
  ↓
ステップ3: 適切な生年月日の範囲設定
  - ストーリーの時点（2026-2027年）から逆算
  - エピソードテーマに適した年齢層
  ↓
ステップ4: accurate-logicで計算・検証
  ↓
ステップ5: 朱学院でダブルチェック
  ↓
ステップ6: ペルソナ特性の生成
```

## 4. 既存キャラクター検証チェックリスト

### 4.1 検証項目
- [ ] 生年月日から逆算した年齢がエピソードテーマに適しているか
- [ ] 天中殺の時期がストーリーと整合しているか
- [ ] 日干（十日干）の性の性格特性が悩みと結びついているか
- [ ] 十大主星・十二大従星がキャラクターの状況を反映しているか
- [ ] 五行バランスが性格傾向と一致しているか

### 4.2 検証プロセス
1. accurate-logicで再計算
2. 朱学院でダブルチェック
3. テーマとの整合性を確認
4. 必要に応じて生年月日を調整
