## 1. キャラクタープロフィール

### 1.1 基本情報

- **名前**: 高橋美咲（たかはし みさき）
- **年齢**: 28歳（2026年時点）
- **誕生日**: 1997年8月14日
- **日柱**: 丁亥（ひのい い）
- **十大主星**:
  - 頭：貫索星（名誉・プライド）
  - 胸：玉堂星（芸術・才能）
  - 右手：調舒星（感受性・繊細さ）
  - 左手：車騎星（行動力・実行力）
  - 腹：調舒星（感受性・繊細さ）
- **五行バランス**:
  - 木22点（少ない）
  - 火32点（十分）
  - 土44点（多い）
  - 金45点（多い）
  - 水53点（非常に多い）
- **天中殺**: 亥子丑

**注意**: 占い師の「星野紫織（48歳）」とは別のキャラクターです。美咲は巡のアシスタントとして物語の中心的な役割を果たします。

### 1.2 性格

**表面的な性格（初期）**:
- 明るく、親しみやすい
- 広告代理店勤務でマーケティングは得意
- SNSでの発信が得意
- 算命学は初心者

**内面的な性格（深層）**:
- 調舒星×2と車騎星の葛藤
- 「行動したいが、感受性が邪魔をする」
- 「自分を表現したいが、繊細さが邪魔をする」
- 「燃え尽きそうな薪」というメタファー
- 転職悩みから巡と出会う

### 1.3 背景

**職歴**:
- 広告代理店に7年間勤務
- SNSマーケティング、インフルエンサーキャンペーンを担当
- 仕事には順調だったが、何か「違う」と感じていた
- 28歳で転職を決意、しかし何をしたいかわからない

**巡との出会い**:
- 第一話で初診患者として来院
- 仕事を辞めるべきか迷う中で巡との対話
- 「木」の重要性を理解し、自分の感受性とSNSスキルを活かす道を探す

**SNS発信の展開（第四話〜第十話）**:
- 第一話での診断体験が「人生を変える」衝撃
- 同じように悩む人に巡の診断を届けたいとSNS発信開始
- 匿名で巡の診断を感動的に発信
- 新規患者の50%以上がSNS経由に
- 自分の行動の影響を知り、再来院（第七話〜第八話）
- アシスタント就任を申し出（第九話〜第十話）

**現在の役割（第十話以降）**:
- 運命診断室のアシスタント
- 広報担当（SNS運営、マーケティング）
- 算命学を学びながら発信している

### 2.3 フェーズ1のまとめ

**美咲の状態**:
- 表面的な理解でSNS投稿をしている
- ラベリングで伝えようとしている
- 読者からの指摘で「良い面」が伝わっていないことに気づく
- 巡の指導で「本質」の重要性に気づく

**美咲の成長**:
- ラベリングの限界に気づく
- 「良い面」「悪い面」の二項対立を超える必要性を感じる
- バランスの重要性に気づき始める

**読者への教育的効果**:
- 十大主星、五行バランスの存在を知る
- 自分の星、バランスに興味を持つ
- しかし「ラベリング」で終わっている

  - 「動きたいけど、どう思われるかな」
  - 「失敗したらどうしよう」
  - 「周りの反応が気になる」

- 左手（車騎星）：行動の実行力
  - 「決めたら動く」
  - 「行動力がある」

- 腹（調舒星）：自己表現の感受性
  - 「自分を表現したいけど、繊細さが邪魔をする」

**「燃え尽きそうな薪」というメタファー**:
- 木（薪）はあるが、尽きそう
- 火（情熱）は燃えているが、不安定
- 土（灰）が溜まり、心が重い
- 金（プレッシャー）で叩かれ、水（感情）で消えそう

### 5.2 葛藤の解決

```
「美咲、調舒星と車騎星の葛藤は、

 車騎星は行動力。

 両方が必要なんだよ。

 感受性があれば、
 他人の気持ちを深く理解できる。
 行動力があれば、
 目標に向かって進める。

 大事なのは、
 バランス。

 感受性を持ちながら、
 行動する。

 行動しながら、
 感受性を保つ。

 そのバランスを学ぶことが、
 美咲の課題であり、成長の機会だ。」
```

**美咲の解決策**:
```
「…わかりました。
 調舒星と車騎星、
 両方を活かすんです。

 感受性を持って、
 他人の気持ちを理解しながら、
 行動する。

 行動しながら、
 自分の感受性を保つ。

 バランスが大事。

 私、そのバランスを学びます。」
```
