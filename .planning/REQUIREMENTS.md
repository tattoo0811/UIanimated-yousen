# Requirements

**Project:** 陰陽五行アプリ リニューアル
**Version:** v1
**Last Updated:** 2026-01-23

---

## v1 Requirements

### UI/UX - カード型インターフェース

- [ ] **UI-01**: スクロールレスなカードベースUIを実装する
  - スワイプでカードを切り替え可能
  - 1画面内で情報が完結
  - react-native-deck-swiper または同等ライブラリ使用

- [ ] **UI-02**: 複数のデザインパターンを提供する
  - 少なくとも3つのデザインテーマ
  - ユーザーが切り替え可能
  - 各テーマは一貫した色・フォント・アニメーション

- [ ] **UI-03**: リッチなアニメーション効果を実装する
  - スワイプ時の遷移アニメーション
  - タップ時のフィードバック
  - 60fpsを維持

### 動画生成 - サーバーサイドRemotion

- [ ] **VID-01**: 9:16縦動画（15-30秒）を生成する
  - サーバーサイドでRemotion使用
  - 1080x1920または720x1280解像度
  - H.264 + AAC エンコード

- [ ] **VID-02**: ユーザーのニックネームを動画に埋め込む
  - 入力された名前を動画内に表示
  - 動的にテキスト挿入

- [ ] **VID-03**: 7秒フック構成を実装する
  - 0-2s: ビジュアルフック
  - 2-5s: パーソナライズ表示
  - 5-15s: 占い結果開示
  - 15-20s: CTA
  - 20-30s: ブランディング

- [ ] **VID-04**: タイピングエフェクトを実装する
  - 文字が1文字ずつ表示される
  - 背景にプリセット画像を表示

- [ ] **VID-05**: 本質→家族→仕事→恋愛→オチの構成でコンテンツを生成する
  - 各セクションで切り替わる演出
  - 最後にオチ（笑い）を入れる

### シェア機能

- [ ] **SHR-01**: ネイティブシェアシートを実装する
  - React-Native-Share-Calendar 使用
  - iOS/Android各プラットフォームのシェアシート表示

- [ ] **SHR-02**: TikTok/Instagramへの直接投稿を可能にする
  - 各アプリのURLスキーム使用
  - 動画ファイルを渡して投稿画面を開く

- [ ] **SHR-03**: カメラロールへの保存を実装する
  - expo-media-library 使用
  - パーミッション処理

### コンテンツ - バズりやすい言葉翻訳

- [ ] **CNT-01**: 専門用語をバズりやすい言葉に翻訳する
  - TikTok風: 短文、エモい、パンチライン
  - YouTube風: ストーリーテリング
  - Instagram風: ビジュアル重視

- [ ] **CNT-02**: 複数トーンに対応する
  - ユーザーがトーンを選択可能
  - 各トーンで異なる言葉遣い

- [ ] **CNT-03**: プリセット画像用の生成プロンプトを提供する
  - 画像生成AI（DALL-E等）のプロンプトを表示
  - ユーザーが自分で生成可能

### 付加機能

- [ ] **EXT-01**: 友達との相性診断を実装する
  - 複数人の陰陽五行を比較
  - 相性スコア表示
  - シェアしたくなる結果表示

- [ ] **EXT-02**: 2026年の年運に特化したコンテンツを提供する
  - 丙午（ひのえうま）の年の特徴
  - 年運に特化した動画テンプレート

## Out of Scope

- リアルタイムチャット機能 — v1では動画生成で十分
- ユーザー間のソーシャル機能 — まずは個人診断に集中
- 動画内のAI画像生成 — プリセット画像+プロンプト提供で対応
- 複数の占い種類 — 陰陽五行に特化
