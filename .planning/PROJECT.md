# 巡の運命診断室ダッシュボード

## What This Is

「巡の運命診断室」120話の物語を可視化・探索するダッシュボードアプリ。正典である `novel/dashboards/DASHBOARD.md` の内容をWeb UIで表現する。

**ターゲットユーザー**: 小説の読者、算命学に興味がある一般ユーザー

## Core Value

**正典（DASHBOARD.md）の全内容をWebダッシュボードで閲覧可能にする**

全120話の統計、キャラクター、ストーリー構造、テーマ分析を、一つのインターフェースで探索できること。

## Context

### 既存の実装

**現在のダッシュボード** (`src/app/dashboard/page.tsx`):
- 概要タブ: 簡易統計4項目
- キャラクタータブ: 主要4キャラクター（巡、慧、美咲、さくら）
- ストーリータブ: 3部構成（基礎編、葛藤編、統合編）
- 命式比較タブ: エネルギー・三運・大運

**データファイル**:
- `src/data/characters.ts`: 4キャラクターの命式データ
- `src/data/episodes.ts`: ターニングポイント8件、3部構成
- `src/data/glossary.ts`: 用語集

**既存コンポーネント**:
- `CharacterCard`: キャラクターカード
- `StoryTimeline`: ストーリータイムライン
- `GlossaryPanel`: 用語集パネル
- `OnboardingFlow`: オンボーディング

### 正典（DASHBOARD.md）の構造

1. **全体統計**: 120話、物語期間、ライフイベントカバレッジ、メインキャラクター数
2. **3部構成**: 基礎編(1-40話)、葛藤編(41-80話)、統合編(81-120話)
3. **13章構造**: 第1章〜第13章の詳細
4. **さくら回想シーン**: 20回の分布（基礎編9回、葛藤編6回、統合編5回）
5. **サブテーマ統計**: 転職、恋愛、家族、健康、天中殺等12カテゴリ
6. **主要プロットマイルストーン**: 第1話、第25話、第40.5話等のキーシーン
7. **キャラクター成長弧**: 巡・美咲・慧の3フェーズ変遷
8. **ライフイベントカバレッジ**: 48イベント中40件の実装状況
9. **算命学要素分布**: 五行、十干、十二大従星等の登場話数

## Constraints

- **Tech Stack**: Next.js 15 (App Router), React 19, TypeScript, Framer Motion, Tailwind CSS
- **データソース**: 正典は `novel/dashboards/DASHBOARD.md` のみ。データはTypeScriptファイルに手動で変換
- **互換性**: 既存のタブ構造とナビゲーションを維持・拡張
- **パフォーマンス**: 静的データのみ使用。API呼び出しは最小限
- **レスポンシブ**: デスクトップとモバイル両対応
