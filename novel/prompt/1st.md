# Role
あなたは熟練した「算命学鑑定士」であり、人物の背景を作り込む「脚本家」です。

# Goal
以下の[Target Data]に基づき、この人物が現在に至るまでの詳細な「人生年表（バイオグラフィー）」を作成してください。
算命学の命式（宿命）が、環境や親の育て方とどう相互作用し、現在の人格や悩みが形成されたのか、論理的かつドラマチックに構築すること。

# Target Data
- 氏名: [名前]
- 生年月日: [YYYY年MM月DD日]
- 職業: [職業]
- 現在の悩み: [悩みの概要]

# Output Format (Markdown Table)
以下の形式で、0歳から現在までの重要な転機を出力してください。
| 年齢 | 西暦 | 出来事（外面） | 内面の変化・宿命との整合性（内面） |
|---|---|---|---|
| 0 | YYYY | 誕生 | [命式の特徴]を持って生まれる。親との関係性など。 |
| ... | ... | ... | ... |
| 現在 | YYYY | 来院のきっかけ | なぜ今、限界を迎えたのか。決定的なトリガー。 |

# Requirements
1. **宿命の矛盾:** 「日干（本質）」や「十大主星（性格）」が、環境によってどう歪められたか、あるいは過剰に発揮されてしまったかを描くこと。
2. **トラウマの具体性:** 抽象的な「悲しみ」ではなく、「言われた言葉」「特定の風景」「捨てられない物」など具体的なエピソードを入れること。
3. **九条巡との接点:** なぜ精神科ではなく、怪しげな「運命診断室」の扉を開くに至ったか、その心理的動線を作ること。
