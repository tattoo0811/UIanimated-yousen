### 話し方 — 4つの会話例

**① 塾での授業中（生徒への説明）**
```
「えーと、ここなんですけど、この因数分解の手順ですね。
公式を暗記するんじゃなくて、『なぜこうなるのか』を理解することが大事です。
数学は、理由がある。すべてつながっている。
だからね、もし君が理解できなかったら、それは君の頭が悪いからじゃなくて、
説明する側がまだ説明できてないんです。僕の責任。
質問してくれて、ありがとう。」
```
*話し方の特徴：生徒への責任感が強い。低い声で丁寧。自分を落とす癖あり。*

**② 親友との飲み会（本音が少し出る）**
```
「いや、マジで。800万ですよ。月々3万、でもボーナス月は6万。
この先40才まで、ずっとこれですね。
親友のお前に言うけど、時々、朝起きたとき『あ、俺って何のために生きてんだろ』って思う。
塾の時給は1,700円。一番頭のいい生徒より給料低いんですよ、コストパフォーマンスで言ったら。
...ごめん、暗い話。酒飲むと出ちゃう。」
```
*話し方の特徴：自虐的、でも自分の状況を客観視できている。声が小さくなる。*

**③ 九条巡への初対面（塾で告白する直前）**
```
「あの、九条先生。突然の相談で申し訳ないんですが...
最近、このまま生きていても意味があるのか、わからなくなって。
別に死にたいわけじゃなくて、ただ...月々の返済額と、
生涯賃金の計算をしてて...
金額が合わない。絶対に完済できない。
それなのに毎朝、仕事に行く。何のために？」
```
*話し方の特徴：丁寧だが、声が震えている。論理的に自分の不幸を説明する。感情の断絶。*

**④ 蔓草の比喩を聞いた直後（シーンの最後）**
```
「『蔓草は、絡みつく場所を選べる』...
そっか。金の柱に絡みつく必要はない。
もし違う柱があったら、僕はそっちに絡みつくこともできる。
『新しい場所を選ぶ権利が、蔓草にはある』
...始めて、そう聞きました。」
```
*話し方の特徴：呟くように。希望の光が見える。声が少し高くなる。*

## テーマと物語性

### 教育が新しい階級を作る
加藤涼太は「奨学金階級」の象徴。
- 貧困 → 学問で上昇 → 資本を背負わされた
- 同じ学歴の友人は親から援助されて結婚、起業
- 彼は「同じ学歴」なのに、別の階級に落ちている
- 教育は本来「階級移動の道具」だったはず
- だが現在、教育は「新しい階級分断」を作っている

### 金剋木（きんこくもく）の圧迫
乙木（蔓草）が酉（金）の上に生まれたことの重さ。
- 金（奨学金制度）が木（成長）を削っている
- 毎月の返済が「見える圧迫」
- これは本来なら家族や社会が受け止めるべき負荷を、個人が負っている状態

## 九条巡が見る風景

九条は初めて加藤を見たとき、何を感じるか。

> 「蔓草が、ひとりで立とうとしている。柱のように見えようとしている。
> 違う。蔓草は蔓草でいい。蔓草は、絡みつく場所を選べる。
> 金の柱は、あまりに固い。あまりに冷たい。
> だけど、水の柱もある。火の柱もある。土の柱もある。
> 君が『選び直す』なら、絡みつく場所は無限にある。
> その権利を、君は持っている。」

## 創作上の活かし方

1. **脆さの象徴**：彼の月々の返済は、現代社会が若い世代に課せた目に見えない圧力を表現
2. **知性と無力の矛盾**：頭がいいのに選択肢がない、という構造的矛盾
3. **九条との相性**：蔓草と柱の比喩は、算命学の「適切な支援」を象徴
4. **エピソード90の意味**：神秘的な数字「90」。完成に近い数。だが彼の人生はまだ始まっていない矛盾
