# 既存患者フォローアップエピソード設計書

> **目的**: 「その場で変わる」への違和感解消のための変化フォロー設計
> **作成日**: 2026-02-16
> **適用範囲**: EP1-79患者の変化追跡

---

## 📋 設計方針

### 基本原則

1. **来院は「変化の始まり」**: 診断室での気づきは、変化の第一歩
2. **時間経過を描写**: メール・街での再会・追加エピソードで変化をフォロー
3. **失敗と再挑戦**: すぐには解決しない悩みもある

### フォローフォーマット

| 形式 | 内容 | 配置方法 |
|------|------|----------|
| メール・手紙 | 来院後の変化を報告 | エピソードの冒頭や合間に挿入 |
| 街での再会 | 偶然の出会いで変化を目の当たりにする | 巡の日常シーンの中に組み込む |
| 追加エピソード | 1-2話の短いエピソード | 本編と独立したショートストーリー |
| 来院そのもの | 再来院 | 新規エピソードとして構成 |

## 📧 メール・手紙形式

### 田中健太（元EP4）── 部下への接し方変化

**送信日**: 来院から3ヶ月後（2026年7月）
**件名**: 先日のご相談の件です

```
巡様

お久しぶりです。元EP4の田中健太です。
先日は運命診断、ありがとうございました。

来院いただいた翌週から、
部下のミスに対する接し方を変えてみています。

「なぜミスをしたのか」ではなく
「どうすれば防げるか」を一緒に考えるようにしました。

最初は部下たちも驚いていたようですが、
先日、若手の新人からこう言われました。

「最近、部長の部屋に行くのが怖くなくなりました」

その言葉を聞いて、
私の「山」が少しだけ柔らかくなったのかもしれない、と。

これからも、自分の命式と向き合いながら
少しずつ変化していきたいと思っています。

また進展があったらご報告します。

田中健太
```

**物語的意義**:
- 戊土（山）が少しずつ柔らかくなる過程
- 「山は動かない」が「山の上を歩ける」ようになる変化
- 読者に「時間が経てば変わる」という希望を提示

### 村田健一（元EP2）── 新しい事業の立ち上げ

**送信日**: 来院から2年後（2028年4月）
**件名**: 新しい会社を立ち上げました

```
巡様

ご無沙汰しております。
元EP2の村田健一です。

もう2年も経つんですね。
先日は拝啓

さて、ご報告があります。

先月、新しい会社を立ち上げました。
「子供の創造性を伸ばす学習塾」です。

巡に言われた言葉を、ずっと覚えています。

「あなたは己土。養分を含む大地だ。
自分の経験を若い人に養分として渡すことができる」

その言葉が種になりました。
当時は「自分はもう使い古された」と思っていました。
でも、大地は何度でも作物を育てることができる。

まだまだ不安はあります。
でも、初生徒が10名入塾しました。
最初の授業で、一人の子供の目が輝きました。
その瞬間、私もまた生まれ変わったような気がしました。

巡との出会いがなければ、
ここまでたどり着けなかったでしょう。

深く感謝しています。

村田健一
```

**物語的意義**:
- 己土（田畑）の性質が発揮された
- 「使い古された」という自己認識の転覆
- 大地は何度でも作物を育てる（再生の物語）

### 大林拓也（元EP6）── 家庭との時間確保

**送信日**: 来院から1年後（2027年4月）
**件名**: 娘の運動会に行ってきました

```
巡様

ご無沙汰です。元EP6の大林拓也です。

娘の運動会に行ってきました。
昨年までは「仕事がある」と言って行っていませんでした。

でも今年は、予定を空けて行きました。
娘がリレーの一走者でした。

ゴールした瞬間、娘が私の方を見て、
大きく手を振りました。

その笑顔を見たとき、
「ああ、これが正解だったんだ」と思いました。

来院いただいた時、巡はこう言いました。
「庚金は切るだけが仕事じゃない。
自分の一番大切なものを守るための刃にもなる」

私の一番大切なものは、
仕事でも業績でもなくて、家族でした。

今は月に一度は必ず家族での時間を設けています。
妻も娘も、私が変わったと言います。

巡との出会いが、私の家族を救ってくれました。

本当にありがとうございました。

大林拓也
```

**物語的意義**:
- 庚金の「守るための刃」という側面の発見
- 仕事vs家庭の二項対立を超えた統合
- 命式は「固定的な運命」ではなく「使い方次第の道具」

## 🏙️ 街での再会シーン

### 草野千穂（元EP7）── 美術館での再会

**場所**: 横浜美術館
**時間**: 土曜日の午後

```
巡は横浜美術館の企画展を見に来ていた。

自分の研究の合間の休息。
キャンバスに向かう画家たちの魂を感じに来たのだ。

階段を上がり、3階の展示室へ。
現代日本画のコーナーで、巡は足を止めた。

一枚の絵の前で、若い女性が立っていた。
20代半ばくらい。
丸い眼鏡の奥の瞳が、真剣にキャンバスを見つめている。

その横顔に見覚えがあった。

「……千穂さん？」

女性が振り返る。
巡の声に、一瞬の戸惑い。そして、微笑み。

「巡先生……！」

元EP7の草野千穂。
3年前に出会った、絵を描くことを諦めかけていた女性。

「お久しぶり。この絵、いいね」

「……ええ。ずっと見ていました」

千穂の視線は、まだ絵に残っている。
その横顔は、3年前とは違う。
自分の表現方法を探して揺れていた不安は消え、
代わりに、確かな言葉を静かに語る作家の顔があった。

「また描き始めたんですか？」

「……ええ。でも、以前とは違います」

千穂は絵を指差した。

「この画家、岩井美咲さん。
私と同じで、いつも「正しい絵」を描こうとしていました。
でも、この絵……」

千穂の言葉は、確かだった。

「この絵は、正しくない。
でも、すごく心に響く。
私も、こういう絵を描きたいと思うようになりました」

巡は絵を見る。
歪んだ遠近法。
混ざらない色。
でも、確かにそこにある「何か」。

「千穂さんの絵も、どこかに展示されていますか？」

「……まだ、です。でも、来年グループ展に出ます。
怖くはないです。
もう、正しくなくていい、と思っていますから」

千穂は巡を見て、小さく笑った。

「巡先生が言ってくれたでしょ。
『絵は言葉です。あなたの言葉を描けばいい』って」

「……覚えてたんですね」

「ええ。忘れるわけがないです」

千穠のポーチから、スケッチブックが見えた。
いつも抱いていたあのスケッチブック。
でも今は、白紙のまま抱えているのではない。

何度も開かれて、
何度も描かれて、
何度も閉じられた。

その厚みが、
彼女が積み上げてきた時間の重さを語っていた。

「応援していますよ」

巡の言葉に、千穠は深く頷いた。

「……はい。また、できたら見せに来ます」

千穠は再び絵に向き直った。
その背中は、3年前より少しだけ大きく見えた。
```

**物語的意義**:
- 「正しさ」からの解放プロセス
- 調舒星の繊細さが、表現力に昇華された
- スケッチブックの変化（白紙→厚み）＝時間の経過

### 灯里奈々（元EP9）── 公園での再会

**場所**: 診療所近くの児童公園
**時間**: 平日の昼

```
診療所からスーパーに向かう道。
途中にある小さな公園。

春の日差し。
ブランコの鎖が、きしむ音を立てている。

巡は足を止めた。
沙汰のない公園に、一人の女性が座っていた。

ベンチに腰を下ろし、
絵本を読み聞かせている。
相手は5歳くらいの男の子二人。

その声に聞覚えがあった。

「……そして、お空の大きな鯨は、
星をいっぱい食べて、満足げに泳いでいったの」

「うわーすごい！」
「もよっと！もよっと！」

子供たちの歓声。
女性は、優しく微笑む。

その横顔。
元EP9の灯里奈々。

「……奈々さん？」

巡に気づいた奈々は、
驚いたように目を開けた。
それから、子供たちに小さく手合わせをして、
ベンチから立ち上がった。

「巡先生……！お久しぶりです」

「こっちこっち。先生なーんだ？」
子供たちが興味深く巡を見ている。

「あ、ごめんなさい。読み聞かせ、途中までで」

「いいえ。こちらこそ、お変わりなく？」

「はい。少しずつですが……」

奈々は子供たちを見て、微笑んだ。

「この子たち、地域の放課後児童クラブの子。
私、今、ボランティアで読み聞かせをしているんです」

「ボランティア……」

「ええ。最初は怖かったんです。
また子供の前に立つのが。
でも、以前みたいに『教師』としてじゃなくて、
ただ『大人』として、子供たちのそばにいる」

奈々の言葉は、静かだ。
でも、そこには確かな明るさがある。

「復職は？」

「……まだです。でも、少しずつ準備をしています。
来年、特別支援学校の非常勤講師をしてみようかなと」

「それは、いいですね」

「ええ。……巡先生との出会いがなかったら、
ずっと引きこもったままだったかもしれません」

奈々は巡を見て、深くお辞儀をした。

「本当に、ありがとうございました」

「何もしてませんよ。奈々さんが、自分で歩いてくれた」

「ええ。……でも、巡先生が扉を開けてくれたんです」

奈々は子供たちの元に戻った。
再び絵本を開き、優しい声で語り始める。

その姿は、
1年前の診療所で震えていた女性とは、別人のようだった。
```

**物語的意義**:
- 天印星(6)の指導力が、教師以外の形で発揮されている
- 「教師として」ではなく「大人として」＝自己受容
- 復職へのプロセス（いきなりではなく段階的に）

### 森川陽菜（元EP3）── 専門学校編

**タイトル**: 「龍は箱から出た」
**時期**: EP109の6ヶ月後（2026年10月）
**場所**: 東京・美容師専門学校

```
美容師専門学校の実習室。
シャンプー台が10台並び、
朝の光が鏡いっぱいに反射している。

陽菜は、モデルの椅子に座っている女性の髪を梳かしていた。
21歳になった陽菜。
かつて名門女子中に押し込められた少女は、
今は美容師を目指す学生だ。

「陽菜ちゃん、ここちょっと切っていい？」
隣の実習台の友人が声をかけてきた。

「え？いいよ。……どれくらい？」

「ほんとに少し。毛先だけ」

「じゃあ、5ミリくらいでいい？」

陽菜のセニングシスが、
光る刃を空気に滑らせた。

「陽菜ちゃん、コントロールできるねー」
「えへへ、ありがとうございます」

友人の冗談めいた言葉に、陽菜は照れたように笑う。
その笑顔には、言葉に出さない自信がある。

休憩時間。
友人たちとランチルームで食事をしている。

「ねえ陽菜ちゃん、
美容師目指したきっかけって何？」

唐突な問い。
友人たちの視線が集まる。

陽菜は一瞬、言葉を詰まらせた。
でも、すぐに静かに語り始めた。

「……私、中学の時、不登校だったの」

友人たちが驚きの表情。

「学校に行けなくて、部屋に籠って、
ずっと絵を描いてた。
自分が何者かわからなくて、
何がしたいのかもわからなくて」

陽菜はフォークを置き、
自分の手を見た。
インクで染まっていたその手は、
今はシャンプーとトリートメントの香りがしている。

「ある時、母に連れられて命式を見に行ったの。
そこで言われた言葉が、私を変えた」

「言われたこと？」

「うん。……『あなたは大樹なんですよ』って」

陽菜の言葉は、静かだ。
でも、そこには確かな響きがある。

「大樹は、箱に入りきらない。
だから、箱から出ればいいって」

「箱……」

「ええ。名門女子中とか、
『ちゃんとした子』にならなきゃっていう、
自分の中の箱」

陽菜は友人たちを見て、微笑んだ。

「美容師って、
誰かの髪を切る仕事だけど、
本当は、誰かの『箱』を開ける仕事なんだなあって思うの」

「箱、開ける？」

「ええ。髪を切って、
その人が自分らしくなるお手伝いをする。
私がされたように」

友人たちは、陽菜の言葉に瞬きした。
そして、誰かが小さく言った。

「……陽菜ちゃん、すごいね」

「え？」

「いや、だから美容師目指すんだって。
ただ髪を切るだけじゃなくて、
そういうことなんだね」

陽菜は照れたように頷いた。

「……まだまだ勉強中だけど、
そうなりたいと思ってる」

その時、実習室から先生の声がかかる。
「陽菜ちゃん、お客様ですよ」

「はい！行ってきます！」

陽菜は立ち上がり、実習室へ向かった。
その背中を見送る友人たちは、
何かを感じていた。

かつての陽菜は、
自分の箱に入り込んで、
酸欠になりそうだった少女だった。

今の陽菜は、
自分から箱を開け、
他人の箱も開けようとしている青年になっている。

龍は、箱から出た。
そして、空へのぼり始めた。
```

**物語的意義**:
- EP109で示された「変化」の具体化
- 美容師＝「箱を開ける仕事」という再解釈
- 甲木の大樹が、他者を支える存在に成長
- 「自分がされたように、他者もする」という巡の教えの継承

### EP109 森川陽菜（21歳）── 詳細シーン構成

**タイトル**: 「龍は箱に入りきらん」
**時期**: 2026年10月（EP3から6年後）
**場所**: 運命診断室

#### シーン構成案

```
【冒頭】

運命診断室のドアがノックされた。

「どうぞ」

ドアが開くと、そこには一人の青年が立っていた。
21歳くらい。
黒いエプロンをつけ、
手にはスケッチブックではなく、
美容師用のセニングシスケース。

巡は、その顔を見て、息を呑んだ。

「……陽菜さん？」

6年ぶり。
かつて母の後ろに隠れるように座っていた少女は、
今は自分の足で立ち、自分の言葉で話しに来た青年だった。

「巡先生……お久しぶりです」

陽菜は笑った。
その笑顔には、
言葉に出さない確かさがある。

「どうやって？」
「母さんから聞きました。
先生がここにいるって」

陽菜は振り返り、
ドアの外にいる女性に手を振った。
母・真紀だ。

「私は外で待ってるわ」
真紀の言葉には、
かつてのような「正しさ」の刃は消えていた。

陽菜は診療室に入り、
 familiarな椅子に座った。
6年前、母に連れられて座った同じ椅子。

「驚きましたよ。
陽菜さんが自分で来るなんて」

「……ええ。
この6年、いろいろあって」

陽菜はスケッチブックを取り出した。
いや、違う。

「……これ、美容師の専門学校の課題作品です」
「絵、ですか？」

「ええ。ヘアデザインのスケッチ。
巡先生に見てもらいたくて」

陽菜が開いたノートには、
驚くほど繊細な線で描かれた髪型のデザインが載っていた。
「龍のような髪」というキャプションがついている。

「……すごいねえ」

「本当ですか？
巡先生には、ちゃんと評価してほしいんです」

陽菜の冗談めいた言葉に、
巡も微笑んだ。

「 evaluation なんてできませんよ。
でも、感じることはあります」

巡はノートをめくりながら、
静かに語り始めた。

「あなたは、甲辰の日を持っています。
大樹の命式。
龍高星が南にある。
探究心と冒険心」

陽菜は静かに聞いている。

「6年前、初めて陽菜さんの命式を見た時、
祖母のさくらが言った言葉が脳裏をよぎった」

陽菜の目が揺れる。

「『龍は箱に入りきらん』って」

「……覚えてますか？」

「ええ。私、ずっと覚えてました」

『場所は外にあるんです。箱の中にはない』って」

自分の6年を語り始めた。

『名門中学に入れたのに、なんで美容師？』って」

陽菜は苦笑いした。

「でも、私は決めてた。
自分の人生で、自分の箱を選ぶって」

「そして、専門学校では？」

「楽しかったです。
しんどい時もありました。
でも、自分が選んだ場所だったから、
頑張れました」

「これ、先日のコンテストで銅賞もらいました」

「モデルさんが、泣いてくれたんです」
```
