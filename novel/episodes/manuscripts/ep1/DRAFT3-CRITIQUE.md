# Draft 3 問題分析 — Draft 4 執筆前の整理

## 1. 冒頭の美しさの欠如

**問題**: 「百七十八円のおにぎりを、昼休みに食べた。鮭。海苔がしなしなだった。」

文学作品の第一行としては生活感が強すぎる。Tapestryのディテールとしては有効だが、読者を「この世界に入りたい」と思わせる引力がない。

**Draft 2の冒頭**: 「四月の東京は、重力がおかしい。」 — これは詩的で、感覚的で、読者の世界認知を一瞬ずらす力がある。

**解決方針**: 冒頭はSomaticモードの美しい一行で始める。Tapestryのディテール（おにぎり、数字、陽菜）は冒頭ではなく、数段落後に配置。読者をまず「感覚」で掴み、次に「生活」で接地させる。

## 2. 診断の流れの完全欠如（最大の問題）

**現状**: 美咲が座る → お茶 → 沈黙 → 悩みを話す → 生年月日 → 巡が読む → アドバイス

**読者が置いてけぼりになる理由**:

### 2-a. 「運命診断」が何か、読者にも美咲にもわからない

美咲は看板を見て入っただけ。「運命診断」という言葉だけで、何をする場所なのか全くわからない。読者も同様。巡が何も説明しないまま「生年月日を教えてもらえますか」と言うのは、コンテクストなしの唐突な質問になっている。

→ **巡は、美咲が落ち着いた後に、この場所が何をするところかを簡潔に説明する必要がある**。ただし巡らしく、事務的な説明ではなく、自然な語りの中で。

### 2-b. インテーク（受付・カルテ）がない

実在する対面式のカウンセリング・鑑定では、必ず最低限の手続きがある：
- 名前を聞く / 記入してもらう
- 生年月日を聞く理由の説明
- 今日どうされましたか（問診）

現在のDraft 3では「名前」すら巡は聞いていない。なのに診断後に「高橋さん」と呼んでいる（L212）。これは論理的に破綻している。

→ **お茶を出した後、巡が簡単なカード（or 紙）を渡して「お名前と生年月日を書いてもらえますか」とするのが自然**。あるいは口頭で聞く流れの中で、美咲が名前と生年月日を答える。

### 2-c. 診断とは何かの「見せ方」

巡が万年暦を開いて何かを計算している間、美咲（=読者）にとっては「何をしているのかわからない」状態。

ep1-first-encounter.md（旧アウトライン）には問診票（Q1-Q5）があった。この仕組み自体は有用。ただし、大規模な受付システム（受付の男性、複数の患者）は現在の設定（赤字ぎりぎりの個人事業）と矛盾する。

→ **巡が手書きのカードを渡す + 万年暦を引く際に「あなたの生まれた日を、暦で読むんです」と一言説明する**。これだけで読者は「ああ、暦から何かを読むのか」と理解できる。

### 2-d. 具体的に欠けている要素

| 欠けている要素 | 現状 | 提案 |
|---|---|---|
| 名前の確認 | 聞いていない | カードに記入 or 口頭で |
| 運命診断の説明 | 一切なし | 巡が一言で説明 |
| 生年月日を聞く理由 | 説明なし | 「暦で読む」程度の説明 |
| 所要時間の目安 | 不明 | 「三十分ほどいただきます」等 |
| 料金 | 不明 | 「初めての方は無料」or 金額提示 |
| 美咲の同意 | なし | 巡の説明に対する小さなうなずき |

## 3. 時間の問題

**現状**: 美咲は「昼休みは一時間しかない」と描写されている。しかし診断が終わると「夕方」になっている（L334-336）。

→ これは明確な矛盾。選択肢は二つ：
- **(A)** 昼休みに来て、午後の仕事を無断欠勤した（社会的リアリティの描写が必要）
- **(B)** 昼休みではなく、退勤後 or 半休を取った設定にする
- **(C)** 昼休みに看板を見つけ、後日（or 退勤後）改めて訪れた

**(A)が最も文学的に面白い**: 昼休みを超えてしまうことが、美咲にとっての「逸脱」＝金がない命式の人間が、初めて日常を「切った」瞬間になる。これ自体がドラマ。

→ ただし、その場合は美咲がオフィスに戻れなくなったことへの言及が必要。

## 4. 巡の人物像の薄さ

**現状**: 巡は「万年筆を走らせている男」として登場し、お茶を淹れ、命式を読み、メタファーで語る。しかし「元外科医」「MedAIで裏切られた」「さくらを喪った」という過去の重みが、ほぼ伝わらない。

→ 第1話で全てを説明する必要はないが、以下の小さな手がかりは必要：
- 手が大きいこと（外科医の手）を美咲が視点で捉える
- 窓の外を一瞬見る仕草（さくらへの呼びかけ）
- 万年暦を開くとき、一瞬だけ手が止まる（この暦が誰のものか）

## 5. 「巡が何も聞かずにお茶を出す」ことの不自然さ

**現状（L108）**: 「巡は何も聞かなかった。奥の棚から茶器を出し……」

飛び込みの来客に、名前も用件も聞かずにお茶を出し始めるのは、いくら穏やかな人でも不自然。これは「文学的な演出」が「リアリティ」を犠牲にしている例。

→ **解決**: 巡が「どうぞ、まずお掛けください」→ 美咲が座る → 巡が「お茶を淹れますので」と言いながら準備 → お茶を出してから、簡単に自己紹介 + この場所の説明。この順序なら自然。

## 7. Draft 4 の構成案

### 第一段（冒頭）: 詩的な導入 + 美咲の世界
- 美しい一行で始める（感覚的・空気的）
- 通勤シーンで数字の世界を描写
- 陽菜との関係、おにぎり、家計簿 → Tapestryディテール

### 第二段: 路地裏 → 看板 → 階段
- 昼休み（or 退勤後）に偶然路地に入る
- 看板を見つける → 迷い → 階段を上る
- ※ここまではDraft 3と同様でOK

### 第三段: 診断室の空間 + 受入フロー（★新規★）
- 扉を開ける → 空気の変化
- 巡の登場（手の大きさ、目、佇まい）
- 「どうぞ」→ 座る → お茶
- **巡が自己紹介 + 診断の説明**（1-2行で簡潔に）
- **名前と生年月日をカードに記入してもらう**
- 美咲が「運命診断って……何をするんですか」と聞く（読者の代弁）
- 巡が答える（算命学用語なしで）

### 第四段: 問診（美咲が悩みを話す）
- カードを書きながら、沈黙
- 巡の「今日はどうされましたか」的な問い（穏やかに）
- 美咲がぽつりぽつりと語る
- 巡は聞く → 待つ → 核心に触れない

### 第五段: 鑑定（巡の内面 + 語り）
- 巡が万年暦を開く → 美咲に「あなたの生まれた日を、暦で読みますね」
- 巡の内面: 数字を読む、五行を数える、さくらとの記憶
- 巡の語り: メタファーのみ、数字なし
- 美咲の反応: 涙ではなく、体の反応

### 第六段: 鉢植え + 水
- 結論部。巡のアドバイス。
- ここは現在のDraft 3が十分に美しい

### 第七段: 帰路（苦い余韻）
- 時間の経過を処理（午後/夕方）
- 草のモチーフ
- 上司からの通知
- 「何も変わっていない」
- 足の裏の汗が引いた ← ミニマルな変化
