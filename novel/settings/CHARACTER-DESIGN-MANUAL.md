## 設計基本原則

### 三つの柱

キャラクター設計では、以下の三つの要素を統合する必要がある：

```
┌─────────────────────────────────────────┐
│         キャラクター設計の三要素          │
├─────────────────────────────────────────┤
│ 1. 物語的多様性                          │
│    - 年齢層のバランス                    │
│    - テーマの網羅性                      │
│    - 悩みのリアリティ                    │
├─────────────────────────────────────────┤
│ 2. 算命学的整合性                        │
│    - 日干・天中殺の正確性                │
│    - 十大主星・十二大従星の分布           │
│    - 五行バランスの多様化                │
├─────────────────────────────────────────┤
│ 3. 物語的一貫性                          │
│    - 主人公との共通点・対比点            │
│    - 再登場キャラクターの成長             │
│    - 算命学的メタファーの貫徹             │
└─────────────────────────────────────────┘
```

### キャラクター設計の七つの鉄則

1. **算命学的正確性第一**: 生年月日から命式を正確に計算する
2. **テーマと命式の整合**: 悩みと算命学的特徴を連動させる
3. **多様性の確保**: 同じパターンの繰り返しを避ける
4. **リアリティの追求**: 具体的な職業・経済状況・家族構成
5. **主人公との接点**: 共通点（共鳴）か対比点（学び）のどちらか
6. **成長の可能性**: 診断後にどう変わるかの示唆
7. **再登場の余地**: サブキャラクターは複数話での展開を想定

## 年齢層別キャラクターテンプレート

### 🌱 10代（15-19歳）: 「形成期の迷い」

**基本特性**:
- 人生の岐路に立つ（進学・進路・アイデンティティ）
- 親からの自立願望と依存の板挟み
- SNSによる比較・承認欲求・いじめ問題
- 算命学的には「エネルギーの高さ」と「制御の難しさ」

**典型的悩みパターン**:

| 悩み | 関連する算命学的特徴 | 事例 |
|-----|---------------------|------|
| 不登校 | 龍高星（型にはまらない）＋エネルギー高 | 森川陽菜（第三話） |
| 進路迷い | 日干の強弱と環境の不適合 | 総エネルギー中程度の割れ |
| いじめ | 天中殺期の自尊心低下 | 寅卯天中殺の思春期 |
| 承認欲求 | 車騎星（行動力）と調舒星（感受性）の矛盾 | SNS中毒・過剰な自己開示 |

**キャラクター作成時のチェックポイント**:
- [ ] 保護者同席か単独来院か
- [ ] 学校との関係性（教師・友人）
- [ ] 総エネルギー値の設定（高いほど制御困難）
- [ ] スマホ・SNSとの関わり方

---

### 💼 20代（20-29歳）: 「選択の連続」

**基本特性**:
- 就職・転職・恋愛・結婚の連続した選択
- 社会人としてのアイデンティティ形成
- 理想と現実のギャップに悩む時期
- 算命学的には「天中殺の影響」が最初の周期で現れる

**典型的悩みパターン**:

| 悩み | 関連する算命学的特徴 | 事例 |
|-----|---------------------|------|
| 職業適合性 | 日干と職業五行のミスマッチ | 木の人が金の業界で消耗 |
| バーンアウト | 五行バランスの偏り | 火の人が木なしで燃え尽きる | 高橋美咲（第一話） |
| 恋愛不安 | 天中殺期の自己価値感低下 | 申酉天中殺の24-25歳 |
| キャリア迷い | 龍高星の探求心と安定の板挟み | 起業志望と親の圧力 |

**キャラクター作成時のチェックポイント**:
- [ ] 就職年数（新人〜5年目程度）
- [ ] 職業の五行分類（参考：職業五行対応表）
- [ ] 恋愛・結婚状況
- [ ] 天中殺の影響有無

**職業五行分類（簡易版）**:

| 五行 | 職業例 | 特徴 |
|-----|--------|------|
| **木** | 教育・医療・福祉・農業 | 育成・成長・創造 |
| **火** | エンターテインメント・IT・飲食 | エネルギー・表現・サービス |
| **土** | 公務員・不動産・建設・金融 | 安定・仲介・蓄積 |
| **金** | 法務・製造・技術・軍警 | 規律・精度・強さ |
| **水** | 運輸・通信・流通・観光 | 移動・柔軟・適応 |

### 🏢 30代（30-39歳）: 「責任の重圧」

**基本特性**:
- 結婚・出産・育児・介護の複合的責任
- キャリア形成と仕事と育児の両立
- 「これでいいのか」の人生問い直し
- 算命学的には「第二の天中殺周期」や「転機の年」

**典型的悩みパターン**:

| 悩み | 関連する算命学的特徴 | 事例 |
|-----|---------------------|------|
| 仕事と育児の両立 | 天将星（リーダー）と家庭五行の矛盾 | 30代での天将星燃え尽き |
| 結婚危機 | 夫婦の五行バランス崩壊 | 水剋火・木剋土など |
| キャリア不振 | 天中殺期の停滞感 | 申酉天中殺の36-37歳 |
| 親の介護 | 干支の相性と親の天中殺 | 親の介護と自分の天中殺が重なる |

**キャラクター作成時のチェックポイント**:
- [ ] 既婚/独身・子供の有無
- [ ] 親の介護状況
- [ ] 企業での役割（管理職〜専門職）
- [ ] 年齢に応じた天中殺の影響

---

### 🏠 40代（40-49歳）: 「中年的危機」

**基本特性**:
- 達成したことへの虚無感
- 親の介護・子供の独立・自身の加齢
- 「人生の後半」への不安
- 算命学的には「大きな転機の年」

**典型的悩みパターン**:

| 悩み | 関連する算命学的特徴 | 事例 |
|-----|---------------------|------|
| 中年の危機 | 天将星のエネルギー減退 | かつての成功体験が再現できない |
| 親の介護 | 干支の相性問題 | 親子の五行関係が悪化 |
| 早期退職願望 | 牽牛星（プライド）と現実のギャップ | 定年前の役職喪失不安 |
| 空の巣症候群 | 子供独立後の五行バランス崩壊 | 育児に捧げたエネルギーの行き場なし |

**キャラクター作成時のチェックポイント**:
- [ ] 管理職・専門職としての地位
- [ ] 親の介護有無
- [ ] 子供の自立状況
- [ ] 定年までの残り年数への意識

---

### 🌅 50代+（50-69歳）: 「再出発と振り返り」

**基本特性**:
- 定年退職・役割喪失・健康不安
- 「生きがい」の再定義
- 孫の世代との関係
- 算命学的には「第三の天中殺周期」と「人生の総括」

**典型的悩みパターン**:

| 悩み | 関連する算命学的特徴 | 事例 |
|-----|---------------------|------|
| 定年後の生きがい | 天将星・牽牛星の役割喪失 | 仕事がアイデンティティだった人 |
| 健康不安 | 五行バランスの崩壊 | 加齢による特定五行の減退 |
| 夫婦関係 | 子供独立後の夫婦五行再確認 | 嫁姑問題解決後の夫婦だけの時間 |
| 孫世代との関係 | 現代価値観との衝突 | SNS時代の孫との接し方 |

**キャラクター作成時のチェックポイント**:
- [ ] 定年退職済/定年前
- [ ] 現役時代の職種・役割
- [ ] 健康状態
- [ ] 孫の有無

## 職業別悩みパターン

### 学生（高校生・大学生）

**共通悩み**: 進路決定・学業成績・友人関係・親との関係

| 学年 | 典型的悩み | 算命学的関連 |
|-----|----------|-------------|
| 高校1-2年 | 部活・勉強・友人のバランス | 総エネルギー高すぎて制御不能 |
| 高校3年 | 進学先選択・親の期待 | 日干と進路五行のミスマッチ |
| 大学1-2年 | サークル・勉強・アルバイト | 初めての自立と天中殺の影響 |
| 大学3-4年 | 就職活動・親との距離 | 職業五行との相性問題 |

### 会社員（正社員・契約社員・派遣社員）

**共通悩み**: 職場の人間関係・昇進・給与・ワークライフバランス

| 職種 | 五行 | 典型的悩み | 算命学的関連 |

| 管理 | 金・土 | 部下育成・責任重圧 | 天将星のエネルギー問題 |

### 自営業・フリーランス

**共通悩み**: 収入不安・顧客獲得・自己管理・孤独

| 職種 | 五行 | 典型的悩み | 算命学的関連 |

## キャラクター作成ワークフロー

### Step 1: エピソード要件の確認

エピソード担当チームから以下の情報を受け取る：

1. **章のテーマ**: 例：「五行深化篇」「天中殺篇」
2. **算命学的ポイント**: 例：「木剋土の関係」「天将星のエネルギー問題」
3. **物語的役割**: 例：「主人公の対比」「再登場キャラクター」

### Step 2: 年齢層とテーマの決定

**年齢層選択フローチャート**:

```
現在の年齢層分布を確認
  ↓
不足している年齢層を特定
  ↓
章のテーマと合う年齢層を選択
  ↓
既存キャラクターとの重複を確認
```

**テーマ設定フローチャート**:

```
章のテーマから悩みのカテゴリを決定
  ↓
年齢層に応じた具体的悩みを選択
  ↓
算命学的特徴と悩みを連動させる
```

### Step 3: 算命学的設計

1. **日干の選択**: まだ少ない日干から選択
2. **天中殺の確認**: 年齢から天中殺の時期を確認
3. **十大主星・十二大従星**: 章のテーマに合う星を配置
4. **五行バランス**: テーマに合う強弱関係を設定

### Step 4: プロフィール作成

以下の項目を設定する：

- **基本情報**: 名前・年齢・職業・生年月日
- **家族構成**: 独身/既婚・子供の有無・親との同居
- **経済状況**: 年収・貯蓄・住宅（持ち家/賃貸）
- **SNS活動**: プラットフォーム・フォロワー数・利用頻度
- **来院動機**: 明確な悩みと背景
- **算命学的特徴**: 日干・天中殺・主要な星・五行バランス
- **主人公との接点**: 共通点（共鳴）か対比点（学び）

### Step 5: 診断シナリオの作成

1. **来院時の状態**: 悩みの具体的な描写
2. **算命学的解釈**: 命式から読み取れる特徴
3. **処方箋**: 具体的な行動提案
4. **変化の示唆**: 診断後の成長の可能性

```yaml
基本情報:
  名前: 九条 巡（くじょう めぐる）
  生年月日: 1990年3月2日
  年齢: 36歳（物語開始時点、2026年4月）

算命学的特徴:
  日柱: 癸卯（水陰）
  天中殺: 申酉（主人公にとって重要）
  十大主星: 龍高星×2・車騎星・禄存星
  総エネルギー: 15点（標準的）
  特徴:
```
