# Japanese Writing Method — 文体モード定義

執筆時は必ず以下のいずれかのモード、またはハイブリッドを指定すること。

---

## 1. Somatic Mode（身体的感性モード）

| 属性 | 内容 |
|------|------|
| **スタイル** | **辻村深月**的アプローチ |
| **特徴** | 感情を「悲しい」「嬉しい」などの抽象語で書かず、「喉の渇き」「視野の歪み」「指先の震え」などの**身体的反応（Somatic Marker）**として描写する |
| **視点** | 極端な内的焦点化。主人公の感情バイアスにより、世界が歪んで見える「信頼できない語り手（Unreliable Narrator）」を許容する |
| **用途** | 回想シーン、トラウマのフラッシュバック、和解のクライマックス |

### 執筆例
- ❌ 「彼は悲しかった」
- ✅ 「彼は飲み込もうとしたが、喉が張り付いて何も通らなかった。視界の端が歪み、向かいの顔が歪んで見えた」

---

## 2. Logic Mode（論理・抑制モード）

| 属性 | 内容 |
|------|------|
| **スタイル** | **米澤穂信**的アプローチ |
| **特徴** | 感情語を徹底的に排除し、論理的な推論や、無機質な**客観的相関物（雨の匂い、時計の音、落下する塵）**へのこだわりによって、逆説的に感情の「空白」を際立たせる |
| **視点** | 制限された一人称、またはシニカルな観察者 |
| **用途** | 診断シーン前半、謎解き、苦い結末（Bitter Aftertaste） |

### 執筆例
- ❌ 「彼女は不安だった」
- ✅ 「彼女は三度、壁掛け時計を確認した。秒針の音が、部屋の乾燥した空気に響いていた」

---

## 3. Tapestry Mode（社会・多声モード）

| 属性 | 内容 |
|------|------|
| **スタイル** | **宮部みゆき**的アプローチ |
| **特徴** | **生活の詳細（家計、食事、職業的苦労）**を緻密に描き、個人の悩みを社会構造（貧困、格差、家族制度）と接続する |
| **視点** | 多声的（Polyphonic）。主役以外の「声」や、噂、報道などの視点をシームレスに織り交ぜる |
| **用途** | 世界観の描写、複数のエピソードの接続、群像劇パート |

### 執筆例
- ❌ 「彼らは貧しかった」
- ✅ 「月末のコンビニ弁当は、税込み438円だった。母はレシートを折りたたみ、財布の一番奥に押し込んだ」

---

## ハイブリッド指定

モードは組み合わせ可能。

- **Somatic + Logic**: 内的独白は身体的に、外的描写は論理的に（例：回想パートの現在進行形）
- **Logic + Tapestry**: 社会構造の分析を論理的に、個人の感情を抑えて描く
- **Somatic + Tapestry**: 社会状況を主人公の身体的反応を通して描く
