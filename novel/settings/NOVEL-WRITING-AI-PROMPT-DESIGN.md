## 第1章: 序論 - 本ドキュメントの目的と設計哲学

### 1.1 目的: AIエージェント向けプロンプト設計書

本ドキュメントは、AIエージェント（Claude/GPTなど）が「巡の運命診断室」を自律生成するためのプロンプト設計書である。対象は120話のオムニバス形式小説で、各話で異なる患者が運命診断室を訪れる構造をとる。

AIへの指示の核心は以下の三点に集約される：

1. **命式システムの物語への翻訳**: 四柱推命・十大主星・十二大従星・天中殺を、キャラクターの悩み・行動原理・成長軌道に変換する
2. **構造的枠組みの適用**: SandersonのPPPモデルとSwainのScene & Sequel理論を、診断シーン形式に適応させる
3. **日本的感性の再現**: 辻村深月・米澤穂信らの視点技術・感情描写・エピソード接続技法を統合する

### 1.2 設計哲学: 日本的感性 × AI最適構造 × 命式システム

**日本的感性の重視**
- 一人称（巡）の語り口：冷静な観察眼・祖母（さくら）から学んだ算命学的洞察・父（精神科医）から学んだ傾聴力（押し付けなアドバイスはしない）
- 感情描写の「見せ方」：身体反応・情景・対比を通した間接的表現
- 視点の技術：診断シーン（巡視点）と回想シーン（患者視点）の多重構造

**AI最適構造の採用**
- PPPモデル（Promise-Progress-Payoff）: 読者の期待形成・進行・報酬を明示的に管理
- Scene & Sequel理論: 診断シーンをGoal-Conflict-Disaster、Reaction-Dilemma-Decisionの6ビートで構造化
- キャラクタースライダー: Likeability・Competence・Proactivityを数値化し一貫性維持

**命式システムの統合**
- 生年月日（immutable）→ 四柱推命 → キャラクター属性（性格・行動原理・悩み）
- 運命＝命（固定）＋ 運（可変）というテーマの構造的表現
- 天中殺・大運と物語の転換点の連動

### 1.3 収集対象作家・技法の選定理由

| 作家 | 採用技法 | 適用範囲 | 理由 |
|------|----------|----------|------|
| 辻村深月 | 視点技術・感情描写・多重構造 | 全エピソード | 「かがみの孤城」等の心理描写精度 |
| 米澤穂信 | エピソード間接続・オムニバス構成 | シリーズ構成 | 「古典部シリーズ」の統一テーマ維持 |
| Brandon Sanderson | PPPモデル・キャラクタースライダー | 物語構造 | AIへの状態遷移モデルとしての親和性 |
| Dwight Swain | Scene & Sequel理論 | シーン設計 | 診断シーンの6ビート構造化 |

## 第2章: 命式システムと物語構造の統合

### 2.1 命式からキャラクター造形への変換

#### 2.1.1 生年月日 → 四柱推命 → 性格・行動原理

**変換フロー**:

```
生年月日（入力）
  ↓
四柱推命計算（年柱・月柱・日柱・時柱）
  ↓
┌─────────────────────────────────────┐
│ 日干（12種）: 基本性格・五行属性      │
├─────────────────────────────────────┤
│ 十大主星（10種）: 行動原理・価値観    │
├─────────────────────────────────────┤
│ 十二大従星（12種）: エネルギー質      │
├─────────────────────────────────────┤
│ 天中殺（6種）: 人生の転機・テーマ     │
├─────────────────────────────────────┤
│ 大運（10年周期）: 人生の章構造        │
├─────────────────────────────────────┤
│ 年運（1年周期）: 人生の章構造        │
└─────────────────────────────────────┘
  ↓
キャラクター属性（出力）
  - 悩みの根本原因
  - 行動パターン
  - 対人関係の傾向
  - 成長の可能性
```

**日干別キャラクター特性（60パターン分類）**:

各日干は6つの日支との組み合わせで、計60通りのキャラクターパターンが存在する。以下に物語生成に直接活用できる詳細分類を示す。

### 2.2 PPPモデル（Promise-Progress-Payoff）の命式適用

#### 2.2.1 約束（Promise）: 命式から導く初期約束

**約束の三層構造**:

```
┌─────────────────────────────────────┐
│  第1層: トーンの約束                  │
│  - 物語全体の雰囲気                  │
│  - 診断室という空間の静謐さ           │
│  - 巡の語り口の落ち着き              │
├─────────────────────────────────────┤
│  第2層: キャラクターの約束            │
│  - 来院者の初期状態（悩み）          │
│  - 命式から導かれる「約束された性格」 │
│  - 変化の可能性の示唆                │
├─────────────────────────────────────┤
│  第3層: テーマの約束                 │
│  - 「運命＝命＋運」                  │
│  - 在り方を変えれば運は変わる         │
│  - 診断は「在り方の処方箋」          │
└─────────────────────────────────────┘
```

**命式別初期約束の例**:

| 命式パターン | 初期約束 | 診断後の約束変更 |
|-------------|----------|----------------|
| 丙火＋車騎星 | 「燃え尽きそう」 | 「火を支える木を見つける」 |
| 甲木＋龍高星 | 「箱に入りきらない」 | 「飛ぶ準備をする」 |
| 庚金＋牽牛星 | 「強さが呪い」 | 「流れる水へ」 |
| 己土＋天将星 | 「育てすぎ」 | 「自分も育てる」 |

#### 2.2.2 進行（Progress）: 診断セッションでの進行

**進行の三段階**:

1. **問診フェーズ（情報の開示）**
   - 来院者の悩みを傾聴
   - 命式の計算・開示
   - 「驚き」の演出: 「知らなかった自分」の発見

2. **分析フェーズ（意味の紐解き）**
   - 命式と悩みの因果関係
   - 天中殺・大運の現在位置
   - 「なぜ今なのか」の説明

3. **処方フェーズ（行動提案）**
   - 具体的な処方箋
   - 「在り方を変える」提案
   - 小さな一歩の示唆

**進行の管理ルール**:
- 各シーンで必ず「情報の前進」がある
- 車輪の空回りを防ぐステルス・テーゼ・パラグラフの挿入
- 読者の期待を裏切らない「道標（Signposting）」

#### 2.2.3 報酬（Payoff）: 「在り方を変える」実現のカタルシス

**報酬の三層**:

1. **認識の報酬**: 「自分を知る」こと自体の価値
2. **変化の報酬**: 処方箋を実践した小さな変化
3. **未来の報酬**: 「運は変わる」という希望

**報酬の「Unexpected and Inevitable」化**:
- 予測不可能性: 読者が予想しない命式の解釈
- 不可避性: 振り返れば「そうだったのか」と納得
- 疑念管理: Red Herrings & Escalationで意表を突く

### 2.3 Scene & Sequel理論 × 命式セッション

#### 2.3.1 Scene（Goal-Conflict-Disaster）: 来院者の問題提示

**診断シーンのScene構造**:

| ビート | 内容 | 命式との連動 |
|--------|------|-------------|
| Goal | 来院者が求めているもの | 日干の欲求（例: 丙火＝認められたい） |
| Conflict | 障害・葛藤 | 天中殺・五行バランスの不調和 |
| Disaster | 診断前の最大の問題 | 人生の危機（バーンアウト・離縁・不登校等） |

**例: 高橋美咲（第1話）のScene**:
- Goal: 「集団の中で自分の役割を見つけたい」
- Conflict: 乙卯の「群れる戦略家」としての仲間意識と、職場での競争・足の引っ張り合い（嫉妬と競争）
- Disaster: 広告代理店での人間関係に疲弊、退職決意・人生の岐路

#### 2.3.2 Sequel（Reaction-Dilemma-Decision）: 診断と変化の決断

**診断シーンのSequel構造**:

| ビート | 内容 | 命式との連動 |
|--------|------|-------------|
| Reaction | 問診に対する巡の分析 | 命式から読み取れる特徴の開示 |
| Dilemma | 来院者の二択 | 「現状維持か」「変化か」 |
| Decision | 処方箋の受容 | 在り方を変える決意 |

**例: 高橋美咲（第1話）のSequel**:
- Reaction: 「あなたは乙卯の『群れる戦略家』。仲間と共に野原を埋め尽くす力があるが、身内での競争に巻き込まれている」
- Dilemma: 「集団に留まるか」「自分の場所を離れて新しい群れを作るか」
- Decision: 処方箋受容・巡のアシスタントとして「新しい群れ」の一員に

### 9.4 ペルソナ設計の原則

#### 原則1: 算命学的正確性

- 全ての生年月日はCLI（`sanmei-with-energy-cli.ts`）で検証
- 日干・天中殺・十大主星・十二大従星は計算結果を正確に反映
- 五行バランスは数値で表現

#### 原則2: 物語との整合性

- 表面の悩みと本当の核（算命学的根拠）を対比
- 天中殺・大運と物語の転換点を連動
- 五行の相剋関係を人間関係に反映

#### 原則3: 巡との接点

- 共通点（共鳴）か対比点（学び）のどちらかを明確化
- 巡の丙火・水0との関連性を意識
- 木生火（美咲→巡）などの五行関係を活用

#### 原則4: 象徴的一貫性

- 元素の象徴（太陽・若草・大河など）を貫通
- 季節・時間帯・色・場所を統一
- 比喩が命式と矛盾しない
