## 1. 医師のキャリアパスと結婚・出産タイミング

### 医師の典型的なライフイベント

```
【標準的な医師のキャリア】

18歳: 医学部入学（6年間）
25歳: 医師国家試験合格・卒業
25〜29歳: 初期臨床研修（レジデント）
    └─ スーパーローテーション
    └─ 不規則な勤務時間
    └─ 結婚・出産は困難

30〜34歳: 後期研修・専門医取得
    └─ 生活が少し落ち着く
    └─ 専門医の資格取得
    └─ 結婚・出産のタイミングとして最適

35歳〜: 開業または勤務医
    └─ 生活が安定
    └─ 子育てとの両立が可能
```

### 旧設定の問題点

```
【旧設定】
父: 1968年生まれ（58歳）
母: 1970年生まれ（56歳）
巡: 1991年生まれ（35歳）

問題点:
└─ 巡出生時、父は23歳・母は21歳
└─ 医師国家試験合格は25歳
└─ つまり、医師になる「前」に結婚・出産している
└─ レジデント時代の多忙な時期に育児というのは現実的ではない

→ 医師として仕事が落ち着いてから結婚・出産という設定に修正
```

## 5. 世代間のギャップと物語的深み

### 三世代の価値観

```
【第一世代】さくら（1925年生）
└─ 戦前・戦中・戦後を生きた世代
└─ 「耐える」「受け入れる」「伝統を守る」価値観
└─ 陰陽道・占いという「非科学的」な世界観
└─ 巡に「時間をかける愛」を教える

【第二世代】篤人・美和子（1950-60年代生）
└─ 高度経済成長を生きた世代
└─ 「成功」「効率」「科学医学」を信じる
└─ 「科学は全てを解決する」という確信
└─ 巡に「期待・プレッシャー」をかける

【第三世代】巡（1992年生）
└─ バブル崩壊後・失われた20年を生きた世代
└─ 「科学医学」に疑問を持つ
└─ 「データでは測れない2%」を求める
└─ さくらの「非科学的」教えに惹かれる

→ 三世代の対立と統合が物語に深みを与える
```

### さくらの「戦争体験」

```
さくらが経験した三つの喪失:

1. 【戦争中の喪失】（1945年・20歳）
   └─ 空襲・食糧難・学徒動員
   └─ 同級生・友人の多くが戦死
   └─ 「命の儚さ」を肌で学ぶ

2. 【夫・寿郎との死別】（1995年・70歳）
   └─ 50年連れ添った夫との死別
   └─ 「愛は離れても消えない」を学ぶ
   └─ 孤独の中で陰陽道の研究に没頭

3. 【最期】（2024年・99歳）
   └─ 18年間育てた孫（巡）が見守る中での逝去
   └─ ノートという形で「遺産」を残す

→ さくらの教えは、全てが「実体験」に基づいている
→ 戦争を生きた強さと、夫を喪った深さがある
```
