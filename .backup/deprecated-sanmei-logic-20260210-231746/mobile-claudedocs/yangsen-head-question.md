# 陽占「頭」の計算に関する質問

## 矛盾点

ユーザー様の説明では：

> 位置 | 使う干
> --- | ---
> 中央（頭） | **日干そのもの**

しかし、1984-12-02の画像データでは：

```
日    月    年
壬    己    甲    ← 干
                  禄存    ← 頭
```

## 計算結果の違い

### パターン1: 日干そのもの（日干×日干）
```
日干=壬 × 日干=壬 = 比和 = 貫索星
```
**現在の実装結果**: 貫索星

### パターン2: 日干×年干
```
日干=壬(水=4, 陽=0) × 年干=甲(木=0, 陽=0)
関係: (0-4+5)%5 = 1 (洩気)
Index: 1*2+0 = 2
結果: TEN_STARS[2] = 鳳閣星
```

### 画像の正解
```
頭 = 禄存星
```

## 質問

「中央（頭）= 日干そのもの」の正しい解釈は：
1. 日干×日干で計算（比和） → 貫索星
2. 日干を基準に年干と比較 → 鳳閣星
3. 別の計算方法 → 禄存星

どれが正しいでしょうか？

## 補足：禄存星を得る条件

禄存星 = 財 = 日干が剋する + 陽干同士

壬(水陽)が剋する五行 = 火
火の陽干 = 丙

しかし年干=甲(木陽)なので、この計算では禄存星になりません。

何か見落としている計算ルールがある可能性があります。
