## HTMLテーブル構造分析

### 人体図テーブルのレイアウト

朱学院の人体図は3行x3列のテーブル構造で表現されています：

```
┌─────────┬─────────┬─────────┐
│         │  頭     │  腹     │  ← 行1
├─────────┼─────────┼─────────┤
│ 左手    │  胸     │  右手   │  ← 行2
├─────────┼─────────┼─────────┤
│ 左肩    │  左足   │  右足   │  ← 行3
└─────────┴─────────┴─────────┘
```

### 各セルのHTML構造

**行1（上部）:**
- [0][0]: 空セル（rowspan=2で中央と結合）
- [0][1]: **頭** - 十大主星
- [0][2]: **腹** - 十大主星

**行2（中部）:**
- [1][0]: **左手** - 十大主星
- [1][1]: **胸** - 十大主星（自分自身）
- [1][2]: **右手** - 十大主星

**行3（下部）:**
- [2][0]: **左肩** - 十二大従星
- [2][1]: **左足** - 十二大従星
- [2][2]: **右足** - 十二大従星

## 部位の意味

### 十大主星（5つの部位）
- **頭**: 親（自分より上の人）
- **胸**: 自分自身（心と思いやり）
- **腹**: 子ども（自分より下の人）
- **右手**: 配偶者（片腕、補佐役）
- **左手**: 兄弟姉妹・友人（自分と同じ立場の人）

### 十二大従星（3つの部位）
- **左肩**: 初年期（親との生活が密接な時期）
- **左足**: 中年期（自分と子どもが縁が深い時代）
- **右足**: 晩年期（夫婦二人きりになる時代）

// 人体図テーブルから星を抽出する関数
function extractStarsFromTable(bodyTable) {
  const rows = bodyTable.rows;

  return {
    tenMajorStars: {
      head: rows[0][1],       // 頭 (右上)
      chest: rows[1][1],      // 胸 (中央)
      abdomen: rows[0][2],    // 腹 (右下)
      rightHand: rows[1][2],  // 右手 (右中)
      leftHand: rows[1][0]    // 左手 (中左)
    },
    twelveMinorStars: {
      leftShoulder: rows[2][0], // 左肩 (下左)
      leftFoot: rows[2][1],     // 左足 (下中)
      rightFoot: rows[2][2]     // 右足 (下右)
    }
  };
}
