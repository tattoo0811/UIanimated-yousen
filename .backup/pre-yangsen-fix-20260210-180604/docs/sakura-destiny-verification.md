## 問題点の分析

### 1. 十大主星計算の不一致

十大主星は日干（日主）から計算される：
- 日主: 戊（土性陽）
- 正しい十大主星パターン:
  - 頭: 牽牛星（戊干→牽牛星）
  - 胸: 玉堂星（戊干×2→玉堂星）
  - 腹: 牽牛星
  - 右手: 調舒星（戊干×3→調舒星）
  - 左手: 石門星（戊干×4→石門星）

accurate-logic は誤って以下を出力：
- 頭: 司禄星
- 胸: 石門星
- 腹: 司禄星
- 右手: 貫索星
- 左手: 龍高星

### 2. 天中殺の不一致

天中殺は陰陽宿命の対冲干支から決まる：
- 朱学院: 辰巳 → 天中殺は戊戌、己卯、乙丑
- accurate-logic: 申酉 → 不正確

陰陽宿命の計算自体に問題がある。

## 結論

1. **accurate-logic の十大主星計算ロジックに重大な問題がある**
   - 日干から十大主星を決定するテーブル/ロジックが誤っている
   - 修正が必要

2. **陰陽宿命の計算も再検討が必要**
   - 朱学院の「辰巳」という結果の計算ロジックを理解する必要がある
   - 日支（戌）と月支（卯）からどうやって「辰巳」になるのか？

3. **採用方針**
   - beads データには朱学院の結果を採用
   - accurate-logic の修正は別途検証チームに依頼
