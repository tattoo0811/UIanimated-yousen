# 大運・年運 詳細鑑定ツール

## 概要

ストーリー創作時、AIが算命学の鑑定レベルで「西方/中央/東方」の関係を参照できるよう、
大運・年運に位相法（干合・支合・対冲・半会・害・刑・破）を付与した構造化データを出力します。

## 使い方

```bash
# 基本: 大運・年運（2020-2040年の年運を含む）
npx tsx tools/taiun-nenun-cli.ts 1990-03-02 male

# 年運の範囲を指定
npx tsx tools/taiun-nenun-cli.ts 1990-03-02 male 2024 2035
```

## 列の意味

| 列 | 説明 | AI創作での活用例 |
|----|------|------------------|
| 西方 | 大運/年運 vs 日支（配偶者）の関係 | その時期の恋愛・結婚・パートナー運 |
| 中央 | 大運/年運 vs 日柱（自分）の関係 | 自己実現・仕事運・転機 |
| 東方 | 大運/年運 vs 年柱（親・社会）の関係 | 親子関係・上司・社会的評価 |
| 天中殺 | 〇＝該当方位が虚の時期 | 無理をしない、準備期間として描写 |

## 関係指標の読み方

- **干合**: 強く引き合う。結婚・パートナーシップに吉
- **支合**: 自然な流れ。協調・発展
- **対冲**: 衝突・転換。変わり目の年として描写
- **半会**: 広がる。可能性の拡大
- **害**: すれ違い。誤解・ストレスに注意
- **刑**: ぶつかり合い。試練・成長の機会
- **破**: 白紙・リセット。新たな始まり
