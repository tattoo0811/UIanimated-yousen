## 計算方法

干支の計算は、立春（2月4日）で年が変わる暦法を使用しています。

```
例）1991年3月20日生まれ
- 立春（2月4日）を過ぎているので、1991年として計算
- 十干: (1991 - 3) % 10 = 戊（つちのえ）
- 十二支: (1991 - 3) % 12 = 辰（たつ）
- 結果: 戊辰（つちのえたつ）
```
