# 中医学陰陽論に基づく九条巡キャラクター再々設計

## 修正のポイント

### ❌ 前回の誤り
- 「身体は陰」「心は陽」という単純化
- 身体を一概に「陰」と決めつけた

### ✅ 正しい理解（中医学・黄帝内経）
```yaml
身体の陰陽（相対的バランス）:
  体表（皮膚・外側）: 陽
  体内（内臓）: 陰
  上部（頭・背中）: 陽
  下部（腹）: 陰
  背: 陽
  腹: 陰

  機能・活動（気・動き・熱）: 陽
  物質・静止（血・津液・精）: 陰

  五臓（肝・心・脾・肺・肾）: 陰
  六腑（胃・胆・小腸など）: 陽

  健康の定義: 「陰平陽秘」（陰が平らかで陽が秘められる）
  原則: 「孤陰不生、独陽不長」（陰だけでは生まれず、陽だけでは育たない）

  結論: 身体全体は陰陽の統一体、分離できない
```

    - 体表を切開（陽的な侵入）

    - 出血（血＝陰の流出）をコントロール

    - 傾聴: 受動的・受容・静 → 陰的行為
    - 対象は心（陰陽両方、しかし活動という面では陽的）
    → 「陽（心の活動）に陰（傾聴）で寄り添う」

    - 静寂な空間で対話（陰的環境）

## 陰陽五行論の正確な理解

### 1. 人体の陰陽（中医学）

```yaml
【身体的陰陽】
  体表（陽）:
    - 皮膚・毛
    - 背中
    - 体の外側
    - 上半身

  体内（陰）:
    - 臓器
    - 腹
    - 体の内側
    - 下半身

  五臓（陰）:
    - 肝: 血を蔵す、怒を主る
    - 心: 神を蔵す、喜を主る
    - 脾: 営を蔵す、思を主る
    - 肺: 気を蔵す、憂を主る
    - 肾: 精を蔵す、恐を主る

  六腑（陽）:
    - 胆: 決断をつかさどる
    - 胃: 受納・腐熟
    - 小腸: 清濁を分ける
    - 大腸: 伝化・糟粕を排出
    - 膀胱: 津液の貯蔵・排泄
    - 三焦: 気の通り道

  物質（陰）:
    - 血
    - 津液
    - 精

  機能（陽）:
    - 気
    - 動き
    - 熱

【健康の定義】
  「陰平陽秘」:
    - 陰が平らか（安定）
    - 陽が秘められる（過剰に放出されない）
    - 陰陽がバランスしている状態

【病気の定義】
  陰陽の失衡:
    - 陰盛陽衰: 陰が強すぎて陽が弱い（寒証）
    - 陽盛陰衰: 陽が強すぎて陰が弱い（熱証）
    - 陰陽両虚: 両方とも弱っている
```

### 2. 五行（木火土金水）と陰陽

```yaml
【五行と臓腑】
  木:

    - 情志: 喜
    - 色: 赤

  土:
    - 五臓: 脾（陰）
    - 六腑: 胃（陽）
    - 季節: 土用（季節の変わり目）
    - 方角: 中央
    - 情志: 思
    - 色: 黄

  金:
    - 五臓: 肺（陰）
    - 六腑: 大腸（陽）
    - 季節: 秋
    - 方角: 西
    - 情志: 悲
    - 色: 白

  水:
    - 五臓: 肾（陰）

    - 季節: 冬
    - 方角: 北
    - 情志: 恐

【五行の相生・相克】
  相生（母から子へ）:
    木 → 火 → 土 → 金 → 水 → 木
    「木は火を生じ、火は土を生じ……」

  相克（制御関係）:
    木 → 土 → 水 → 火 → 金 → 木
    「木は土を克し、土は水を克し……」

  健康とは:
    五行がバランスよく相生・相克している状態
```
