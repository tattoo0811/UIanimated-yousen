## 陰陽思想の考証

### 1. なぜ「陰陽」なのか？（「陽陰」ではない理由）

#### 東洋思想における順序の哲学

**「陰陽」の語順の意味**:
```
陰陽（いんよう）: 陰が先、陽が後
陽陰（よういん）: 陽が先、陰が後
```

**なぜ「陰陽」が正しいのか**:

1. **易経（えききょう）の原理**
   ```yaml
   易経:
     太極 → 両儀（陰陽） → 四象 → 八卦
     「易は陰陽を生ず」: 陰陽は対等で同時に生まれる

   しかし:
     「陰が陽を包む」: 形（陰）がエネルギー（陽）を内包する
     「陰が基礎、陽が機能」: 物質的な陰が基盤
   ```

2. **算命学における陰陽**
   ```yaml
   算命学の陰陽:
     陰: 見える形、物質、受動、静、内、冷、夜、月、女
     陽: 見えない気、エネルギー、能動、動、外、熱、昼、日、男

     基本原理:
       「陰陽互根（いんようごこん）」: 陰陽は互いに根を共有
       「陰陽消長（いんようしょうちょう）」: 増減しながらバランス
       「陽陰転化（よういんてんか）」: 極まれば転じる

     なぜ「陰陽」か:
       1. 形（陰）が先にあって、機能（陽）が後
       2. 身体（陰）が基盤で、気（陽）が作用
       3. 観察（陰）が先にあって、診断（陽）が後
   ```

3. **東洋医学の陰陽表裏**
   ```yaml
   東洋医学:
     「陰は裏、陽は表」: 内臓（陰）と皮膚・筋肉（陽）
     「陰は血、陽は気」: 物質とエネルギー

     診療の順序:
       1. 望診（見る）→ 陰的観察
       2. 聞診（聞く）→ 陰的受動
       3. 問診（問う）→ 陽的相互作用
       4. 切診（触れる）→ 陽的接触

     「まず陰（観察）して、陽（診断）する」
   ```

**結論**:
- **「陰陽」が正しい**: 観察（陰）→ 診断（陽）、身体（陰）→ 気（陽）の順序
- **「陽陰」は不自然**: エネルギー先、物質後という唯物論的・西洋的

### 2. 外科医×心療内科の陰陽配置

```yaml
【昼間：外科医】
  属性: 陽（表・動・外・熱）
  対象: 身体（物質・形・陰的）
  行為: 手術（能動・侵入・陽的）

  ⇕ 陰陽の相互作用 ⇕

【夜：心療内科】
  属性: 陰（裏・静・内・冷）
  対象: 心（エネルギー・無形・陽的）
  行為: 傾聴（受動・受容・陰的）
```

**この配置の陰陽論的意味**:

1. **身体（陰）への陽的アプローチ（外科）**
   - 身体は「形」= 陰
   - 手術は「動」= 陽
   - **陰に対して陽で働きかける** = 陰陽調和

2. **心（陽）への陰的アプローチ（心療内科）**
   - 心は「気・エネルギー」= 陽
   - 傾聴は「静・受容」= 陰
   - **陽に対して陰で受け止める** = 陰陽調和

3. **昼夜の陰陽**
   - 昼 = 陽（活動・外向）
   - 夜 = 陰（休息・内省）

**九条巡は「陰陽のバランスを生きる医師」**:
- 昼間は陰（身体）に陽（外科）で働く
- 夜は陽（心）に陰（傾聴）で寄り添う
- 24時間で陰陽の循環を完結している
