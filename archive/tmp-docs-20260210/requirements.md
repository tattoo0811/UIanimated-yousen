# 要件定義書: 高橋美咲のSNS投稿例とマーケティング戦略

## 1. プロジェクト概要

### 1.1 目的
高橋美咲（運命診断室のアシスタント兼広報）が運営するSNSアカウントの具体的な投稿内容と、運命診断室のマーケティング戦略を作成する。

### 1.2 背景
- 美咲: 29歳、元広告代理店、丁亥日、調舒星＋車騎星
- 転職悩みから巡との出会いで運命診断室に就職
- 広告代理店出身なのでマーケティングは得意
- 算命学はまだ学び始め（表面的な理解から始まる）

## 2. 機能要件

### 2.1 SNS投稿例の作成（FR-001）

#### FR-001-1: 表層的解説投稿（初期・美咲の投稿）
**優先度**: 高
**説明**: 美咲が算命学を表面的に理解している段階でのSNS投稿

**具体的要件**:
- 貫索星＝「わがままな星」
- 車騎星＝「攻撃的な星」
- 調舒星＝「繊細な星」
- 禄存星＝「金運の星」
- その他の星の表層的解説

**数量**: 5-7件

**各投稿に含む要素**:
- プラットフォーム（Instagram / Twitter / TikTok）
- 文面（キャプション）
- ハッシュタグ
- 画像・動画の説明
- エンゲージメント（いいね・コメント・シェア）
- 読者への教育的効果（表層的）

#### FR-001-2: 深い理解投稿（巡の解説・美咲が学ぶ）
**優先度**: 高
**説明**: 巡の深い洞察を学んだ後の、より正確なSNS投稿

**具体的要件**:
- 貫索星＝「自己主張の星・バランスが重要」
- 車騎星＝「行動力の星・目的があれば力になる」
- 調舒星＝「感受性の星・他者を深く理解する」
- 禄存星＝「安定の星・土台作りの重要性」
- その他の星の深い理解

**数量**: 5-7件

**各投稿に含む要素**:
- プラットフォーム（Instagram / Twitter / TikTok）
- 文面（キャプション）
- ハッシュタグ
- 画像・動画の説明
- エンゲージメント（いいね・コメント・シェア）
- 読者への教育的効果（深い理解）

#### FR-001-3: マーケティング・トピック投稿
**優先度**: 中
**説明**: 運命診断室のサービス紹介と差別化

**具体的要件**:
- 運命診断室の紹介
- 巡のプロフィール
- 患者の匿名化事例
- 占いの違い（算命学 vs 他の占い）
- その他のマーケティングトピック

**数量**: 3-5件

**各投稿に含む要素**:
- プラットフォーム（Instagram / Twitter / TikTok）
- 文面（キャプション）
- ハッシュタグ
- 画像・動画の説明
- エンゲージメント（いいね・コメント・シェア）
- 読者への教育的効果

### 2.2 マーケティング戦略の具体化（FR-002）

#### FR-002-1: 認知拡大戦略
**優先度**: 高
**説明**: SNSを通じた運命診断室の認知拡大

**具体的要件**:
- SNSプラットフォーム選定（Instagram, Twitter, TikTokの比較）
- ターゲット層定義
- コンテンツカレンダー
- 投稿頻度
- ハッシュタグ戦略

#### FR-002-2: 差別化戦略
**優先度**: 高
**説明**: 競合との差別化ポイントを明確化

**具体的要件**:
- AI占いアプリ（星みてる）との違い
- 伝統的な占い師との違い
- 「運命診断」独自の価値提案
- 美咲の「学びながら発信」アプローチの独自性

#### FR-002-3: 数値目標
**優先度**: 中
**説明**: マーケティング活動の具体的なKPI

**具体的要件**:
- フォロワー数
- エンゲージメント率
- 問い合わせ数
- 来院者数

#### FR-002-4: キャンペーン例
**優先度**: 中
**説明**: 具体的なマーケティングキャンペーン

**具体的要件**:
- 「あなたの運命、診断します」キャンペーン
- 「五行バランス診断」キャンペーン
- 「天中殺克服」キャンペーン
- その他のキャンペーンアイデア

### 2.3 美咲の成長物語（FR-003）

#### FR-003-1: フェーズ1: 表面的理解（第一話〜第十話）
**優先度**: 高
**説明**: 美咲が算命学を表面的に理解している段階

**具体的要件**:
- 初期のSNS投稿（表層的解説）
- 読者からの指摘・コメント
- 巡からの指導・訂正
- 美咲の気づき・学び

#### FR-003-2: フェーズ2: 学びと成長（第十一話〜第三十話）
**優先度**: 高
**説明**: 美咲が巡の深い洞察を学び始める段階

**具体的要件**:
- 巡の深い洞察に触れる機会
- 自分のSNS投稿を修正
- 読者からのフィードバックの変化
- 理解が深まるプロセス

#### FR-003-3: フェーズ3: 深い理解（第三十一話〜最終話）
**優先度**: 高
**説明**: 美咲が独自の解説を展開できる段階

**具体的要件**:
- 独自の解説を展開
- 巡との対話・深い議論
- 読者への教育的効果
- プロフェッショナルとしての成長

### 2.4 3者関係の可視化（FR-004）

#### FR-004-1: 巡・慧・美咲の関係性
**優先度**: 中
**説明**: 3人の関係性と役割分担を明確化

**具体的要件**:
- 美咲がSNSで発信することによる、運命診断室の認知拡大
- 慧のAIアプリ「星みてる」との対比
- 美咲が巡の「深い洞察」を理解するプロセス
- 読者も一緒に「学ぶ」構造

## 3. 非機能要件

### 3.1 品質要件
- **NFR-001**: 算命学的整合性を厳守
- **NFR-002**: 実際にSNSで使える自然な文面
- **NFR-003**: プラットフォームごとの特性を考慮

### 3.2 コンテンツ要件
- **NFR-004**: 絵文字・ハッシュタグを適切に使用
- **NFR-005**: 読者に「学び」を提供する構造
- **NFR-006**: 広告代理店出身の美咲らしいマーケティング感

### 3.3 制約事項
- **NFR-007**: 美咲の性格設定（丁亥日、調舒星＋車騎星）を反映
- **NFR-008**: 巡の哲学（表面的なラベリングへの批判）を反映
- **NFR-009**: 慧との対比（AI vs 人間の洞察）を反映

## 4. データ要件

### 4.1 入力データ
- 算命学の基本概念（十二運星、五行など）
- 美咲の性格設定（丁亥日、調舒星＋車騎星）
- 巡の哲学・診断アプローチ
- 慧のAIアプリ「星みてる」の仕組み

### 4.2 出力データ
- SNS投稿例: 15-20件
- マーケティング戦略文書
- 美咲の成長物語
- 3者関係図

## 5. 成功基準

### 5.1 機能的完全性
- [ ] 15-20件のSNS投稿例が作成されている
- [ ] 各投稿に必要な要素（プラットフォーム、文面、ハッシュタグなど）が含まれている
- [ ] マーケティング戦略の4つの要素（認知拡大、差別化、数値目標、キャンペーン）が具体化されている
- [ ] 美咲の成長物語の3フェーズが描かれている
- [ ] 巡・慧・美咲の関係性が明確化されている

### 5.2 品質基準
- [ ] 算命学的整合性が保たれている
- [ ] SNSプラットフォームの特性が反映されている
- [ ] 美咲・巡・慧の性格設定が反映されている
- [ ] 読者に教育的価値を提供する構造になっている

## 6. 用語集

| 用語 | 定義 |
|------|------|
| 算命学 | 中国古来の運命学。十二運星・五行・陰陽などを使う |

| 調舒星 | 十二運星の一つ。感受性・共感を表す |
| 車騎星 | 十二運星の一つ。行動力・推進力を表す |
| 天中殺 | 人生の転機を迎える時期 |
