# 九条巡キャラクター再設計提案

## 現状設定の課題分析

### データ矛盾の解消
- JSONファイル: `age: 26`
- Markdownファイル: `35歳、元精神科医5年`
- **問題**: 年齢設定が一貫していない

### 「厚み」の不足
1. **精神科医5年** = 専門医としての経験値が薄い
   - レジデント2年 + 専門医3年 = ぎりぎり独り立ち
   - 臨床での深い洞察や人間理解が描写不足

2. **転身動機が希薄**
   - 「薬では治らないもの」だけでは抽象的
   - 何を体験してそう思ったのか？具体的エピソード不足

3. **陰陽バランス未活用**
   - 「医師から算命師」だけでは単なるキャリアチェンジ
   - 昼夜の対比、身体×精神の二元性が活きていない

【転身の引き金】

  - 救急処理で身体は完全回復
  - 退院2週間後に再自殺、今度は死
  - 遺書発見：「身体は治ったけど、私の痛みは誰も理解してくれなかった」

  期間: 3年間の修業を経て、開院

【キャラクター性】
  性格:
    - 昼間は: 冷徹な判断、的確な指示、チーム医療の要
    - 夜は: 内省的、問いを持ち続ける、患者の語りに深く耳を傾ける
  口調:
    - 若い患者へ: 「お前」「〜だ」「〜しろ」（年上らしく）
    - 同世代へ: 「君」「あんた」「〜だ」
    - 年上へ: 「あなた」「〜ですね」
  闇:
    - あの少女を救えなかった罪悪感
    - 医師を辞めたことへの周囲の非難
    - 「占い師」に対する世間の軽蔑
  光:
    - 救急現場で培った「生死の境界線」への造詣
    - 500人以上の死現場での学び（死の瞬間に見えるもの）
    - 身体の声なき声を聞く能力

【セリフの特徴】
  - 命令形で簡潔に（救急現場の癖）
  - 間を多用する（精神科医時代の傾向）
  - 身体感覚の比喩（「心が折れる」「根が浮く」）

**この設定の強み**:

- ✅ 転身動機に具体的な悲劇エピソード（少女の死）
- ✅ 35歳での転身がリアル（経験と若さのバランス）
- ✅ 「運命を知ることで自殺を減らす」という社会的使命

## 推奨設定の理由

### 【案A】陰陽バランス極振り型が最も強力な理由

1. **物語的葛藤の深さ**
   - 救命救急医として「身体を救うが、死は防げない」現実
   - 精神科医として「心を診るが、運命は変えられない」限界
   - 算命師として「運命を知り、予防する」第三の道

2. **陰陽の象徴性**
   - 陽: 救命救急（顕在的、身体的、緊急性）
   - 陰: 精神科（潜在的、精神的、慢性的）
   - 統合: 算命学（顕潜在の調和、予防治療）

3. **キャラクターの奥行き**
   - 表: 冷徹な元救命救急医
   - 中: 問い続ける元精神科医
   - 深: 運命を読む算命師
   - 闇: 救えなかった少女への罪悪感
   - 光: 次は救えるという希望

4. **セリフへの影響力**
   - 救急現場経験→簡潔な命令形
   - 精神科経験→間を取る傾聴力
   - 算命師として→運命的断定の重み
   - 三重の経験が言葉に「重層性」を与える
