### 1. 基本情報

```json
{
  "id": "character_id",
  "name": "氏名",
  "name_reading": "よみがな",
  "age": 年齢,
  "gender": "性別",
  "occupation": "職業",
  "family_structure": "家族構成"
}
```

### 2. 悩みの深層

```json
{
  "surface_worry": "表層的な悩み",
  "deep_worry": "深層的な悩み",
  "root_cause": "根本原因",
  "life_event": "関連するライフイベント",
  "emotional_state": "感情状態"
}
```

### 3. 算命学的要素

```json
{
  "rokujukkoushi": "六十星士",
  "nikkan": "日干",
  "yangsen": "陽占",
  "twelve_stars": "十二星",
  "five_elements": "五行"
}
```

### 4. SNS投稿（悩みの表現）

```json
{
  "platform": "X（Twitter）",
  "content": "投稿内容",
  "timestamp": "投稿日時",
  "likes": "いいね数",
```

## 次のステップ

1. **キャラクターデータの作成**: 第1優先の5キャラクターについて詳細データを作成
2. **算命学的計算**: 各キャラクターの六十星士・陽占・十二星・五行を計算
3. **エピソード構成**: 各キャラクターの来院エピソードを構成
4. **SNS投稿の作成**: 各キャラクターのX（Twitter）投稿を作成
5. **プロンプトのテスト**: Grokプロンプトを実際に使用し、リアルな意見を収集

## 注釈

- **既存キャラクター**: 現在の96エピソードに登場する患者キャラクター
- **未登場**: 新規作成が必要なキャラクター
- **対応エピソード**: 該当するエピソード番号（特定困難な場合は「未定」）
- **追加・深化が必要な設定**: 既存キャラクターの場合は深化内容、新規の場合は作成内容
