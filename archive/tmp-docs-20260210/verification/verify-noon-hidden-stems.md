# 午（馬）の蔵干データ検証

## 問題
テストケース4（1995-09-12）で、ユーザー確認値では右手が「調舒星」になるはずだが、計算結果は「貫索星」になっている。

## 分析

### 現在の二十八元データ（mobile版）
```typescript
'午': {
    extra: { stem: '丙', days: 10 },  // 余気: 丙 (10日)
    sub: { stem: '己', days: 9 },     // 中気: 己 (9日)
    main: { stem: '丁' }              // 本気: 丁 (残り)
}
```

### テストケース4の状況
- 生年月日: 1995年9月12日
- 節入り: 白露（9月8日）
- 経過日数: 5日目
- 選択された蔵干: **丙**（余気期間内）

### 計算結果
- 日干: 丙（火・陽）
- 日支蔵干: 丙（火・陰）
- 関係: 同五行・異陰陽 → **貫索星**（比和）

### 期待値
- 右手: **調舒星**（洩気）

## 調舒星になる条件
日干が丙（火・陽）の場合、調舒星（洩気）になるには:
- 異五行かつ異陰陽
- 水（壬・癸）または 金（庚・辛）
- かつ陰（癸・辛）

つまり、蔵干が**癸（水・陰）**または**辛（金・陰）**である必要がある。

## 可能性の分析

### 可能性1: 二十八元データが誤っている
午の蔵干が「丙・己・丁」ではなく、別の組み合わせである可能性。
- 伝統的な三才占学では、午の蔵干は「丁・己」とされていることが多い。
- しかし、水（壬・癸）や金（庚・辛）が含まれるケースは稀。

### 可能性2: 節入日の計算が誤っている
- 現在: 白露は9月8日
- 実際: 1995年の白露は9月8日で正しい
- 経過日数: 9月12日 - 9月8日 + 1 = 5日目で正しい

### 可能性3: ユーザー確認値に誤りがある
- ユーザーが確認した占いサイトで、異なる計算方法を使用している可能性。
- または、ユーザーが別の生年月日で確認した可能性。

### 可能性4: 別の計算ロジックが存在する
- 陰陽道の流派によって、蔵干の順序や期間が異なる可能性。
- 二十八元ではなく、別の蔵干選択方法を使用している可能性。

## 結論
1. **二十八元データの検証**: 午の蔵干が「丙・己・丁」で正しいか、専門書で確認が必要。
2. **ユーザー確認値の再確認**: テストケース4の生年月日と右手が「調舒星」になるか、ユーザーに再確認が必要。
3. **別の計算ロジックの調査**: 陰陰道の流派によって、異なる計算方法がないか調査が必要。

## 追加調査が必要な項目
1. 三才占学の専門書で、午の蔵干を確認
2. 複数の占いサイトで、1995年9月12日の命式を照合
3. ユーザーにテストケース4の詳細を確認
