## 📋 検証プロセス

### フェーズ1: データ収集（待機中）
4つのデザインチームが以下のファイルを作成するのを待機：
1. `EPISODES-1-24-CHARACTERS.json`
2. `EPISODES-25-48-CHARACTERS.json`
3. `EPISODES-49-72-CHARACTERS.json`
4. `EPISODES-73-96-CHARACTERS.json`

### フェーズ2: 個別検証
各キャラクターについて以下を検証：
1. **日干の整合性**（Critical）
   - 設計値と計算値が一致するか

2. **天中殺の整合性**（Critical）
   - 設計値と計算値が一致するか

3. **五行バランスの整合性**（Warning）
   - 許容誤差±5%以内か

4. **エネルギー点数の整合性**（Warning）
   - 許容誤差±3点以内か

### フェーズ3: 統計検証
全キャラクターについて以下の統計を解析：
- **日干分布**: 甲乙丙丁戊己庚辛壬癸（各10個前後）
- **天中殺分布**: 子丑、寅卯、辰巳、午未、申酉、戌亥（各16個前後）
- **年齢層分布**: 10代、20代、30代、40代、50代+のバランス
- **職業の多様性**: 30種類以上の職業があるか
- **性別分布**: 男性・女性のバランス

### フェーズ4: 問題検出と修正
- **Criticalエラー**: 日干・天中殺の不一致 → デザインチームに修正依頼
- **Warning**: 五行・エネルギー点数の多少の誤差 → 許容範囲内か検討

### フェーズ5: 最終レポート作成
`/claudedocs/SANMEIGAKU-VERIFICATION-REPORT.md` に最終レポートを出力

## ⚠️ 重要事項

1. **矛盾はしないように生年月日と占い結果だけはデリケートに**
   - 生年月日から計算した結果と、設計された占い結果が一致するか検証
   - 不一致の場合、どちらを修正すべきか慎重に検討
   - デザインチームのクリエイティビティを尊重

2. **検証はキャラクター設計の完了を待って開始**
   - 無理に急かさない
   - 全4チームの作業完了を待ってから一括検証

3. **軽微な誤差は許容**
   - 五行の点数が±5%程度の差異はOK
   - エネルギー点数が±3点程度の差異はOK
   - 日干・天中殺の不一致のみ重大な問題として扱う
