# めぐり96話エピソード設計方針 v2

## 基本方針

### 設計思想
- **現代のリアリティ**: 2025-2026年の日本社会を反映
- **多様なキャラクター**: 年齢、職業、背景の多様性
- **具体的な悩み**: 抽象的な悩みではなく、具体的なエピソード
- **経済的リアリティ**: 給与、家賃、奨学金、推し活支出の具体化
- **AI感の排除**: クリシェを避け、自然な対話と感情の動き

## キャラクター設計ルール

### 名前の選択
- 同じ名字は全96話中最大5回まで
- 年齢と名前の整合性（例：32歳なら1992-1994年生まれ）
- 珍しい名字を30%程度混入
- 国際的な名前を5話程度
- 性別中立の名前を適宜使用

### 年齢構成
- 10代後半（18-19歳）：5話以上
- 20代前半（20-24歳）：10話以上
- 20代後半（25-29歳）：10話以上
- 30代：20話以上
- 40-50代：20話以上
- 60代以上：10話以上

### 職業の構成
- デジタル・クリエイティブ職：15話以上
- テック・AI関連職：10話以上
- ギグ经济体・フリーランス：10話以上
- 非正規雇用：10話以上
- 伝統的職業：残り

### 経済状況の具体化
- 給与・年収：年齢層別の中央値を使用
- 家賃：東京23区平均10万634円（2025年）
- 奨学金：20代の適宜反映（平均300-400万円）
- 推し活・サブスク：若年層の適宜反映
- 貯金：年齢層別の中央値

## エピソード設計ルール

### 1話あたりの構成
```markdown
## 第X話「タイトル」

| 項目 | 内容 |
|---|---|
| **テーマ** | 算命学の概念 |
| **哲学** | エピソードの核心となるメッセージ |
| **患者** | 氏名、年齢、職業、経済状況 |
| **六十花甲子** | 例：甲子(01) |
| **陰占** | 日干、月支、天中殺 |
| **陽占** | 中心星、四方の星 |

**患者プロフィール**:
- 氏名：〇〇 △△（読み方）
- 年齢：00歳
- 職業：具体的な職業
- 経済状況：年収、家賃、貯金、借金など
- 家族構成：独身、既婚、子供の有無など
- SNS：Twitter/Instagram/TikTokの有無とフォロワー数

**来院のきっかけ**:
具体的なエピソード（どのような出来事があって来院したか）

**訴える悩み**:
具体的な悩み（抽象的な表現を避ける）

**診察プロセス**:
1. 問診：巡と患者の対話
2. 命式診断：陰占と陽占の読み解き
3. 処方箋：具体的なアクションプラン

**バイラルカット**:
SNSで拡散される映像的シーン

**処方箋**:
「処方：〇〇」の形式で、具体的で実行可能なアドバイス

**エピローグ**:
来院から1ヶ月後の変化
```

### セリフの書き方
- **長台詞を避ける**: 1人のセリフは最大3行
- **説明臭くしない**: 説明的な会話を避ける
- **感情の動き**: 段階的な感情の変化を描く
- **リアルな反応**: 初対面で核心を明かさない

### 悩みの具体化
- **抽象的**: 「将来が不安」
- **具体的**: 「26歳で年収300万円。同期は350万円。差が広がるのが怖い」

### 治療のリアリティ
- **即効性を避ける**: 1回の診察で人生が変わらない
- **段階的改善**: 小さなアクションの積み重ね
- **再訪の可能性**: 必要に応じて再訪エピソード
