# センシティブ表現調整＆炎上対策戦略

## トーン＆マナー基本方針

### 四重の安全基準
1. **表現の安全**: 間接的・象徴的表現で直接的描写を避ける
2. **倫理の安全**: 感情的操作を避け、客観的視点を維持
3. **医学的 safety**: 専門的リアリズムで誤解を防ぐ
4. **社会的 safety**: スティグマを強化しない、適切なヘルプメッセージ

---

## 【代替表現案】自殺の直接的描写を避ける戦略

### レベル分けによる表現調整

#### ❌ レッド（禁止表現）
```
- 「自殺した」「自殺未遂」の直接的な繰り返し
- 方法の描写（飛び降り、飛び込み、等）
- 血の描写、遺体の描写
- 詳細な状況描写（時間、場所、動機の具体的記述）
```

#### ⚠️ イエロー（要注意表現）
```
- 「二度と戻ってこられなかった」（偶使用可）
- 「向こう側へ行ってしまった」（偶使用可）
- 「人生を諦めてしまった」（OK）
- 「生きる光を失くした」（OK）
```

#### ✅ グリーン（推奨表現）
```
- 「命を落としてしまった」
- 「もうこの世界にはいない」
- 「救うことができなかった」
- 「二度笑顔を見ることができなかった」
- 「私が気づけば、もう遅かった」
```

## 【転身エピソードの修正案】 softened drama

### バージョンA：救急医の葛藤（推奨）

```yaml
【事件の描写 - 間接的表現】
  年齢: 32歳
  患者は17歳の少女

  巡の記憶（回想シーン）:
    「夜の救急医療現場は、暗い海のような場所だ。
     色々な人たちが、水曜の夜に流れてくる。

     ある雨の夜、17歳の少女が運ばれてきた。
     私は全力で手当をした。身体は元通りになった。
     彼女の母親は『ありがとう、先生』と泣いていた。

     ……しかし。

     私が彼女の心を治すことはできなかった。
     彼女が来院したのは、『身体を治すため』じゃなかった。
     『誰かに気づいてほしかった』という、最後の叫びだったのかもしれない。

     二週間後、私は彼女の母親から電話をもらった。
     電話の向こうから、母親の悲鳴のような声が聞こえた。

     私は救急医として、身体を救った。
     でも、彼女を救うことはできなかった。」

  問い:
    「医師は『身体』を治す。でも『人生』を治すことはできないのか？」
    「彼女の運命を、私は知ることはできなかったのか？」
    「知っていたら、何か変わっていたのか？」
```

**ポイント**:
- ✅ 直接的な「自殺」「方法」の記述を一切排除
- ✅ 「二週間後」「母親の電話」で事実を示唆
- ✅ 医師としての「限界」に焦点を当てる
- ✅ 問い（哲学的問いかけ）を強調

### バージョンB：複数例の積み重ね（より現実的）

```yaml
【複数のケースに触れることで単一の悲劇を回避】

  巡の記憶:
    「救命救急の現場で、私は500人の命と向き合ってきた。
     そのうちの何人かは、自分から『命を投げ出した』人たちだった。

     20代の男性、手当すれば生きられた。
     でも、退院後、彼は二度と戻ってこなかった。

     40代の女性、身体は完治した。
     でも、彼女の心の傷は癒えなかった。

     私は『身体』を救急医として治す。
     でも、彼らが本当に求めていたのは、身体の治療じゃなかった。

     彼らは『気づいてほしかった』のだ。
     誰かに、自分の痛みを。
     私には、それが見えなかった。」

  転機:
    ある夜、当直明けに本院の屋上へ。
    夜明けの空を見ながら、自問する。

    「医師とは何か？」
    「私が救いたいのは、身体か？ 人か？ 人生か？」

    その時、祖父の遺品があったのを思い出す。
    「算命学」— 運命を知る学問。

    「もし、運命を知っていたら……
     彼らの苦しみを、少しでも早く気づけたかもしれない。
     予防できたかもしれない。」

    35歳、医師を辞める。
```

**ポイント**:
- ✅ 単一のドラマチックな悲劇を避ける
- ✅ 救急医としての現実（複数ケース）に基づく
- ✅ 「屋上で夜明けを見る」という象徴的シーン
- ✅ 問いを深める知的アプローチ

## 炎上対策チーム構成

### チームS：センシティブ表現検証チーム
**役割**: 全96話の表現チェック

```yaml
チェック項目:
  1. 自殺の直接的描写
     - 「自殺」「自殺未遂」の多用
     - 方法、場所、状況の具体的記述
     → 間接表現に書き換え

  2. 感情的操作の検出
     - 読者の感情を過度に揺さぶる表現
     - 悲惨さの強調
     → 客観的描写に調整

  3. 医学的正確性の検証
     - 精神科医療の誤解を招く表現
     - 薬物療法への不当な批判
     → 専門的なニュアンスを維持

  4. スティグマ強化の回避
     - 「精神疾患＝危険」の図式
     - 「治る＝強くなる」の単純化
     → 多様性を尊重する表現
```

### チームF：炎上シミュレーションチーム
**役割**: 想定される批判を予測し対策

```yaml
想定批判パターン:

  1. 「医療の否定」批判
     批判: 「医師を辞めて占い師とは現代離れしている」
     対策:
       - 第1話で巡の葛藤を深く描く
       - 「医師としての限界」は普遍的テーマであることを示す
       - 慧との対立で「医学×算命学」の共存を模索する物語に

  2. 「軽い気動で転身」批判
     批判: 「35歳で医師辞めるのは現実的じゃない」
     対策:
       - 5年間のダブルライフ（救命救急＋精神科）を描写
       - 経済的苦境、周囲の反対を描く
       - 算命学修業の厳しさを描写

  3. 「占いの科学性」批判
     批判: 「占いを科学のように描くのは誤解を招く」
     対策:
       - 「予言」ではなく「診断」という言葉を使う
       - AIとの対立で「確率×運命」のテーマに深みを持たせる
       - 「運命は変えられる」という哲学を強調

  4. 「メンタルヘルスの扱い」批判
     批判: 「深刻な問題を簡易に解決している」
     対策:
       - 処方箋は「魔法の解決策」ではなく「行動の提案」
       - 失敗例も描く（第10章以降で）
       - 専門機関への紹介を描写
```

### チームA：表現調整チーム
**役割**: セリフレベルでの表現調整

```yaml
調整例:

  【Before】
    「彼女は自殺した。私が気づけば、もう遅かった」

  【After】
    「彼女は、もうこの世界にはいなかった。

  【Before】
    「お前、自殺する気か？」

  【After】

     ……君の命式には、もっと違う使い道があるんだ」

  【Before】
    「処方：自殺を考えないようにすること」

  【After】
    「処方：今夜、一度だけ、自分に問うこと。
     『自分の人生を、誰のために生きているのか？』と」
```

## 第1話での導入方法（修正案）

### 導入シナリオ

```markdown
### 【プロローグ】巡の独白

（夜の運命外来。手書きの命式図が壁に貼られている）

巡（35歳、白衣）：
「人の命には、重さがある。
 私は12年間、医師としてその重さを抱えてきた。

 救命救急の現場で、身体を治す医師だった。
 夜は心療内科で、心に耳を傾ける医師だった。

 でも、私は気づいてしまった。

 身体は治せる。心は診れる。
 でも、『運命』は変えられない。

 ……いや、変えられないんじゃない。
 知らないだけなんじゃないか？」

（万年筆を置く）

「医師は『結果』に向き合う。
 運命を知る者は、『原因』に向き合う。

 私は、医師を辞めた。
 運命を知るために。

 ここは、運命外来。
 薬は処方しない。
 処方するのは、あなた自身の運命。」

---
```

**ポイント**:
- ✅ 12年間という経験年数を明示（22-34歳）
- ✅ ダブルライフを示唆
- ✅ 「結果」vs「原因」の対比で医師と算命師の違いを明確化
- ✅ センシティブな具体例は後の章（第10-11章）で回想として提示

## 推奨設定（最終案）

```yaml
九条巡:
  年齢: 35歳（12年間の医師歴）
  経歴:
    - 22-27歳: 救命救急医レジデント
    - 27-30歳: 救命急センター医師
    - 30-32歳: 救命急＋心療内科ダブル勤務（夜間のみ）
    - 32-34歳: 精神科医として勤務
    - 34-35歳: 算命学修業
    - 35歳: 「運命外来」開院

  転身動機:
    「12年間で、私は何人もの『身体』を救ってきた。
     でも、彼らの『人生』を救うことはできなかった。

     特にある夜のことは、忘れられない。
     17歳の少女。身体は元通りになった。
     でも、彼女は二度と笑わなかった。

     医師は『結果』としての死を防ぐ。
     でも、人生を諦める『原因』は、治せない。

     ……できるはずだ。
     運命を知ることで、人生を諦める前に、
     何かできるはずだ。」

  信念:
    「運命は、変えられる。
     知ることが、変える第一歩。
     私は、予防したいんだ。
     誰かが、人生を諦める前に。」

  キャラクター性:
    - 表: 冷徹な元救命救急医（簡潔な命令形）
    - 中: 問い続ける元精神科医（間を取る傾聴）
    - 深: 運命を読む算命師（重みのある言葉）
    - 闇: 救えなかった人々への想い
    - 光: 次は救えるという希望

  セリフ特徴:
    - 若い患者へ: 「お前」「〜だ」「〜しろ」
    - 同世代へ: 「君」「あんた」「〜だ」
    - 年上へ: 「あなた」「〜ですね」
    - 共通: 処方箋は「〜しろ」「〜を受け入れろ」
```
