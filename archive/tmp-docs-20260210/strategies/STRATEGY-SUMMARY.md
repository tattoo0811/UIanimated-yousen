## 3. 組み込み戦略

### 基本原則

1. **既存エピソードの尊重**: 元のテーマを変更しない
2. **多層的な悩みの表現**: 一人のキャラクターが複数のライフイベントに直面
3. **算命学的整合性**: 追加ライフイベントも陰陽五行・算命学で説明可能

### 組み込み案件の詳細

#### 案件1: 第27話「天貴星＋天恍星」× E05: 大学受験

**キャラクター**:
- 山本大輔（19歳、2浪目）
- 父：山本剛（48歳）
- 母：山本敏子（48歳）

**追加プロット**:
```
大輔は2浪目。予備校寮に住み、親への罪悪感と合格への焦燥感。
敏子は「何度かかってもいい」と言うが、内心は親戚への顔向けを気にしている。

巡は三人の命式を見る。
大輔：甲午日干、天貴星（高い理想）
「天貴星は高い理想を持つ。だが理想が高すぎるからこそ、現実とのギャップに苦しむ。
浪人は失敗じゃない。天貴星の高い理想に向かう、必要な修行期間だ」
```

**算命学的解釈**:
- 天貴星の高い理想×現実のギャップ
- 2浪目の意味：焦りではなく、熟成期間

#### 案件2: 第76話「親の介護」× F05: 出産・育児、M03: うつ病、X05: 自己破産

**キャラクター**:
- 伊藤弘子（48歳、介護離職）
- 母：伊藤フミ（82歳、認知症）
- 長男：伊藤翔太（15歳、不登校）

**追加プロット**:
```
弘子は母の介護のため、45歳で会社を辞めた。
ある日、浴槽で母の身体を拭いている最中に泣き崩れる。
「私は何をしているんだろう。誰のために生きているんだろう」

診察室で、弘子は初めて自分の状況を語る。
「貯金はもう底。夫は『介護施設に入れればいい』と言うが、
母を施設には送れない。でも、このままでは家計が崩壊する。うつかもしれない」

巡は静かに弘子の掌を見る。
「壬水だ。大海のように全てを受け入れる性質。
だが大海も、受け入れすぎれば汚染される。
あなたは母を受け入れ、翔太を受け入れた。その結果、あなた自身が溺れている」
```

**算命学的解釈**:
- 壬水の包容力が枯渇するプロセス
- 水（介護）が洪水になり、自分自身を飲み込む

## 4. 拡張戦略

### 基本戦略

**96話→120話の段階的拡張**:
- 第1フェーズ：96話→105話（優先度の高い9イベント）
- 第2フェーズ：105話→120話（中優先度の15イベント）

### 第1フェーズ：97-105話

#### 第97話「内定零の秋」 - E06: 就職活動

**テーマ**: 丙午(43)・丙火の燃え尽きと自己肯定感

**キャラクター**:
- 小林奈緒（22歳、大学院生、就活中）
- 兄：小林真一（24歳、既卒、内定零）
- 母：小林幸子（52歳）

**あらすじ**:
```
奈緒は就活開始が遅れ、内定ゼロ。
「自分には何の魅力もない。誰にも選ばれない」

巡は奈緒の命式を見る。
「丙午。太陽だ。
丙火は周囲を照らす。だが自分を照らせない。
内定零は、あなたの価値が低いからじゃない。
雲（自分を否定的に見る癖）が厚いからだ」
```

**処方箋**:
```
「処方：太陽は自分を照らせない。だが太陽は雲の向こうにも輝いている。
内定零は、あなたの価値が低いからじゃない。雲が厚いだけ。
週に一度、研究でも就活でもなく、ただ自分のために輝く時間を作れ」
```

#### 第98話「婚活パーティーの星」 - F01: 婚活

**テーマ**: 甲己合・結婚相性と干合の縛り

**キャラクター**:
- 吉田明日香（33歳、婚活中）
- 母：吉田恵子（60歳）

**あらすじ**:
```
明日香は婚活アプリ5社利用。
「条件が高すぎる」と言われるが、「妥協したくない」

巡は明日香の命式を見る。
「甲木だ。大樹になる性質。
干合は『魅力的な関係性』を示すが、『結婚すべき関係性』を示すわけじゃない。
まずは、自分という大樹を育てろ。その根が深ければ、誰かが来る」
```

#### 第99話「干合の罠」 - F07: 不倫・浮気

**テーマ**: 乙庚合・強烈な相性と不倫

**キャラクター**:
- 妻：宮下美咲（35歳）
- 夫：宮下誠一（40歳）
- 不倫相手：岡本莉子（28歳）

**あらすじ**:
```
美咲は夫の浮気を発覚。子供のために離婚せず、信頼回復の苦悩。
莉子は既婚者と知りながら恋愛、別れられない依存。

巡は三人の命式を見る。
「庚と乙は干合（乙庚合）。強烈な相性。
だが干合は磁石のような引力。引き合うが、触れてはいけない。
その引力に従えば、家庭という土台が崩壊する」
```

## 5. 陰陽五行・算命学の観点からの多角化

### 五行バランスの多様性

**既存**:
- 木不足：転職、キャリア迷い
- 火不足：燃え尽き症候群
- 土不足：家族問題、介護
- 金不足：リストラ、経済的困難
- 水不足：孤独、孤立

**新規パターンの追加**:
- 木火土金水の組み合わせによるライフイベントへの影響
- 三合会局による人間関係の強制力とライフイベント
- 方冲による対立関係とライフイベント

### 天中殺の網羅性

**現状**: 6種の天中殺は網羅済み

**課題**:
- 天中殺中のライフイベント対応が不十分
- 天中殺とライフイベントの因果関係が不明確

### 十大主星・十二大従星

**現状**: 網羅済み

**課題**:
- 現代的ライフイベントとの統合が不十分
- キャリア、家族、経済的課題との関連が浅い

## 6. 戦略的提言

### 組み込み（既存96話への追加）

**メリット**:
- エピソード数を増やさずにテーマ追加
- キャラクターの深掘り
- 算命学的統合理解

**注意点**:
- 既存テーマとの整合性維持
- 読者の理解の負担に注意
- キャラクターのリアリティ維持

### 拡張（新規エピソードの追加）

**メリット**:
- ライフイベント網羅性の向上
- 読者エンゲージメントの向上
- アプリ診断への導線強化

**注意点**:
- ストーリーの一貫性
- 算命学的整合性
- 制作コストの管理
