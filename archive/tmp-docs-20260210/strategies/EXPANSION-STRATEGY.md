# 拡張戦略（新規エピソードの追加）

**更新日**: 2026-02-09
**目的**: 新規エピソード追加によるライフイベント網羅性向上

---

## 拡張の基本戦略

### 1. 96話→120話の段階的拡張
- **第1フェーズ**: 96話→105話（優先度の高い9イベント）
- **第2フェーズ**: 105話→120話（中優先度の15イベント）

### 2. 新規エピソードの設計原則
- **明確なライフイベント**: 一つの主要ライフイベントにフォーカス
- **算命学的深掘り**: 陰陽五行・算命学からの明確な解釈
- **既存キャラクターとの接続**: 既存エピソードやキャラクターとの関連性
- **読者共感度**: 「自分ごと化」を促す具体性

### 3. エピソード構成の維持
- 既存の11章構造を尊重
- 新規エピソードは「第十二章：現代篇」として追加
- または「第九章：人生の岐路篇」の拡張

### 第97話「内定零の秋」 - E06: 就職活動

**テーマ**: 丙午(43)・丙火の燃え尽きと自己肯定感

**キャラクター**:
- **主人公**: 小林奈緒（22歳、大学院生、就活中）
- **兄**: 小林真一（24歳、既卒、内定零）
- **母**: 小林幸子（52歳、就活失敗の責任を感じる）

**あらすじ**:
```
奈緒は大学院生だが、就活開始が遅れ、内定ゼロ。
「研究を続けたい」という親の期待と「就職しなければ」という現実の板挟み。
兄の真一は既卒扱いで面接すらもらえず、家に閉じこもる。

幸子は二人の子供の就職を心配し、
「私の教育が間違っていたのか」と自責する。

診察室に来たのは奈緒。
「自分には何の魅力もない。誰にも選ばれない」
「研究でも、仕事でも、何をやってもうまくいかない」

巡は奈緒の命式を見る。
「丙午。太陽だ。
丙火は周囲を照らす。だが自分を照らせない。
あなたは研究という『誰かの役に立つこと』に逃げ込んでいた。
自分を輝かせることを恐れていた」
```

**算命学的解釈**:
- 丙火の太陽的性質
- 自己肯定感の低下（雲に隠れる太陽）
- 新卒一括採用の日本独特のプレッシャー

**処方箋**:
```
「処方：太陽は自分を照らせない。だが太陽は雲の向こうにも輝いている。
内定零は、あなたの価値が低いからじゃない。
雲（自分を否定的に見る癖）が厚いからだ。
週に一度、研究でも就活でもなく、ただ自分のために輝く時間を作れ。
その時間が、雲を晴らす」
```

**バイラルカット**: 雨の日のキャンパス。内定ゼロのスマホ画面。その奥で微かに輝く太陽。
キャッチ「内定零は、あなたの価値じゃない。雲が厚いだけ」

- **主人公**: 吉田明日香（33歳、婚活中）
- **母**: 吉田恵子（60歳、娘の結婚を焦る）
- **婚活相手**: 佐藤太一（35歳、年収400万）

**あらすじ**:
```
明日香は婚活アプリ5社利用、婚活パーティー多数参加。
「条件が高すぎる」と言われるが、
「妥協したくない」と思う。
親から「早く結婚しろ」と急かされ、自己嫌悪が悪化。

診察室で、明日香は初めて自分の弱さを告白する。
「結婚したいのは、一人が怖いからかもしれない。
誰かと人生を歩む意味を見出せていない」

巡は明日香の命式を見る。
「甲木だ。大樹になる性質。
大樹は一人で立つ。だが大樹も、根は土と繋がっている。
結婚は『土を見つけること』じゃない。
『自分という大樹を、誰かと共有する』ことだ」

巡は干合の図を描く。
「甲と己は相性がいい。だが相性がいいから結婚できるわけじゃない。

『結婚すべき関係性』を示すわけじゃない」
```

**算命学的解釈**:
- 甲木の大樹的性質
- 干合（甲己合）の魅力的だが危険な関係
- 結婚と相性の違い

**処方箋**:
```
「処方：干合は『魅力的な関係性』を示す。だが結婚は別だ。
甲の大樹は、自分の根を張ってから、誰かと共有する。
結婚は『一人が怖いから』するんじゃない。
『この人と、大樹を共有したい』と思うからするんだ。
まずは、自分という大樹を育てろ。その根が深ければ、誰かが来る」
```

**バイラルカット**: 婚活パーティーで笑顔を作る女性。その影で疲れ切った表情。
キャッチ「婚活は、自分という大樹を育てる過程」

### 第99話「干合の罠」 - F07: 不倫・浮気

**テーマ**: 乙庚合・強烈な相性と不倫

**キャラクター**:
- **妻**: 宮下美咲（35歳、夫の浮気発覚）
- **夫**: 宮下誠一（40歳、浮気をした）
- **不倫相手**: 岡本莉子（28歳、既婚者と知りながら恋愛）

**あらすじ**:
```
美咲は夫の浮気を発覚。子供のために離婚せず、信頼回復の苦悩。
誠一は「二度としない」と誓うが、美咲は心を閉ざす。

莉子は既婚者と知りながら恋愛、別れられない依存。
「私だけが彼を理解している」と思い込む。

診察室に来たのは美咲。
「彼を許したい。でも信頼を取り戻せない。
離婚すれば、子供が傷つく。離婚しなければ、私が傷つく」

巡は三人の命式を見る。
「誠一さんは庚金。美咲さんは甲木。莉子さんは乙木。
庚と乙は干合（乙庚合）。強烈な相性。
だが干合は『魅力的な関係性』を示すだけで、
『正しい関係性』を示すわけじゃない」

巡は不倫の構造を説明する。
「干合は磁石のような引力。引き合うが、触れてはいけない。
その引力に従えば、家庭という土台が崩壊する。
莉子さんは乙の蔦。庚金という幹に絡みつくことで生きている。
だがその幹は、別の誰か（美咲さん）の幹だ」
```

**算命学的解釈**:
- 乙庚合の強烈な引力
- 干合の危険性（既婚者同士）
- 乙木の蔦的特性

**処方箋**:
```
「処方：干合は磁石のような引力。だが引力に従うべきじゃない。
誠一さん、庚金の刃は、家庭という鞘に入っている。
その鞘を外せば、刃は錆びる。
美咲さん、甲木の大樹は、風に揺れても倒れない。
あなたの根は深い。その根を信じて、許す勇気を持て。
莉子さん、乙の蔦は、自分の根を張れる幹を探すんだ」
```

**バイラルカット**: 夫婦と不倫相手を三角形で表現。干合の引力線。
キャッチ「干合は引力。でも重力に従う必要はない」
