# キャラクター表現の「バズり」ルール (gemini.md)

このドキュメントは、四柱推命や干支などの伝統的なキャラクター要素を、SNSで拡散されやすく、共感を生む「バズり表現」に変換するためのガイドラインです。

## 1. 構成の5つの柱（バイラル・エンジニアリング）

視聴者の認知リソースを瞬時に獲得するため、以下の5つの「共感ロジック」で構成します。

| 項目 | 説明 | バイラル化のコツ |
| :--- | :--- | :--- |
| **キャッチコピー** | そのキャラのアイデンティティを記号化した異名 | 「執行人」「攻略不可能」「ラスボス」など。0.5秒で認識できる言葉。 |
| **バズり表現** | 内面のマルチタスクや脳内のバグ | 「脳内リツイート再生」「感情のフロー制御」など。 |
| **強み・ポテンシャル** | アテンションを獲得する圧倒的性能 | 「神すぎる」「レべチ」「最短距離ハック」など。 |
| **負の側面（弱点）** | 現代社会における「垢BAN」や「フリーズ」 | 「人生の地図からブロック」「裁きのスイッチ」「心のシャッター全閉」。 |
| **行動パターン** | 視聴者に「既視感（デジャヴ）」を抱かせる具体例 | SNS、職場、深夜のネットなど、特定のシチュエーションに落とし込む。 |

---

## 2. 表現の変換パレット（アテンション・エコノミー対応）

伝統的な占星術の抽象表現を、ターゲットの「ドーパミン的反応」を誘発するパワーワードに置換してください。

### ① 「デジタル・メタファー」の活用
*   **Before:** 絶交する、忘れる
*   **After:** 人生の地図から消去（ブロック） / 垢BAN / 履歴デリート
*   **効果:** 現代のデジタル・コミュニケーション空間における関係断絶を記号化する。

### ② 「精密センサー・脳内バグ」
*   **Before:** 感受性が強い、考えすぎ
*   **After:** 超精密センサーが微細なノイズを増幅 / 脳内リツイート再生が止まらない / 情報過多でフリーズ
*   **効果:** 現代人の「マルチタスクの疲れ」への強烈な共感を呼ぶ。

### ③ 「戦略的演出（ギャップ）」
*   **Before:** 明るい、強い
*   **After:** 表は完璧なエンターテイナーだが、裏で「一人反省会」 / 穏やかな顔の下で虎視眈々と牙を研ぐ
*   **効果:** 視聴者に「自分だけの秘密を暴かれた」というデジャヴ感を与える。

### ③ 「性能・センサー」化
*   **Before:** 感受性が強い、細かい
*   **After:** 超精密センサー / 情報過多で脳がパンク / 相手の裏を秒で読み取る
*   **効果:** 性格を「能力」や「仕様」として定義することで、特別感を演出する。

### ④ 「中二病」の回避とバランス調整
カッコつけすぎた（中二病的な）表現が強すぎる場合は、日常的な「あるある」や「癒やし」の言葉に引き戻します。

| 中二すぎる表現 | バズりと日常のバランス調整例 | 効果 |
| :--- | :--- | :--- |
| 魂を救うヒーラー | **「職場にいたら神」な癒やし系** | 精神世界から「現実社会」へ引き戻す |
| 闇（ダークサイド） | **実は「一人が大好き」なだけ** | 深刻さを減らして共感しやすくする |
| 裁きのスイッチ | **「あ、無理」ってなる瞬間** | 誰にでもある日常の感覚に落とし込む |
| 革命の指導者 | **みんなが諦めた時に動く人** | 大げさな役割を「具体的な行動」にする |

### ⑤ 「バイラル・キーワード」の適正使用
*   **ハック** → 「攻略法を見つける」「裏技」
*   **デリート（消去）** → 「さらっと忘れる」「なかったことにする」
*   **フリーズ** → 「考えすぎて止まる」「思考停止」
*   **アテンション（注目）** → 「ついつい目立っちゃう」

## 3. 陽占ハック・ライフデザイン

陽占の5方向を、現代の人間関係やライフステージで見える「顔」に変換します。

| 座標 | 伝統名 | バイラル名 | 意味・あるある |
| :--- | :--- | :--- | :--- |
| **中央** | **主星** | **「自分（コア）」** | 本音、一人になった時の本質、自分を動かすマインド。 |
| **北** | **目上** | **「背中（ルーツ）」** | 上官（上司）との付き合い方、理想、目指す背中。 |
| **南** | **目下** | **「アウトプット」** | 部下・後輩・子供への接し方、表現。 |
| **東** | **社会** | **「顔（パブリック）」** | 仕事モード、初対面、社会への出方、友人。 |
| **西** | **家庭** | **「内側（プライベート）」** | 配偶者への顔、家でのリラックス法（スイッチOFF）。 |

## 4. 出力テンプレート（陽占ハック版）

各キャラクターは以下の3つのセクションで構成します。

### [番号] [干支名]：[バイラル・キャッチコピー]
#### ①【本質】コア・スタイル
*   **バズり表現**: そのキャラの本質を突いた一言（例：人生を最短ハックする賢者）
*   **強み/弱点**: 圧倒的なポテンシャルと、誰もが共感する「脳内バグ」。

#### ②【外向き】社会・上下の顔
*   **上司（北）**: どんな上司と合うか、理想の追いかけ方。
*   **部下（南）**: 後輩からどう見られるか、アウトプットの癖。

#### ③【内向き】関係・プライベート
*   **社会（東）**: 仕事での立ち回り、友人との距離感。
*   **家庭（西）**: 家で見せる素顔、唯一スイッチOFF（リフレッシュ）できる場所。
