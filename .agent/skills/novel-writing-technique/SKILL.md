# Novel Writing Technique Skill

このスキルは、3つの特殊な「文学的モード」を使い分け、キャラクター設計、プロット構築、シーン執筆を行うための専門ツールです。

## Literary Modes (文学的モード)

執筆時は必ず以下のいずれかのモード、またはハイブリッドを指定すること。

1.  **Somatic Mode (身体的感性モード)**
    *   **スタイル**: 辻村深月的アプローチ。（参考: 『かがみの孤城』『ゼロ、ハチ、ゼロ、ナナ。』『鍵のない夢を見る』）
    *   **核心原理**: 視点人物の精神状態が世界の描写そのものを歪ませるほどの「極端な内的焦点化」。客観的事実よりも、登場人物が互いの痛みを理解し合えるかという**「感情的真実（Emotional Truth）」**を優先する。
    *   **特徴**: 嫉妬・自己嫌悪・承認欲求など直視したくない感情を容赦なく言語化し、それを「呼吸が浅くなる」「喉が渇く」「手足が震える」「視界が歪む」といった**身体的反応（Somatic Marker）**として克明に描写する。「悲しい」などの感情ラベルは一切使用しない。
    *   **視点**: 極端な内的焦点化。思春期の自意識過剰や劣等感のフィルターを通して世界を語る「感情バイアスによる信頼できない語り手」を許容する。主人公の思い込みを地の文に織り込み、事実を歪める。
    *   **用途**: 回想シーン、トラウマのフラッシュバック、和解のクライマックス。

2.  **Logic Mode (論理・抑制モード)**
    *   **スタイル**: 米澤穂信的アプローチ。（参考: 『氷菓』〈古典部〉シリーズ — 「日常の謎」の確立者）
    *   **核心原理**: 事象の構造や因果関係は精緻に言語化するが、生々しい感情については意図的に記述を避ける。小さな謎から**帰納的論理**を積み重ね、その論理の完成図が予想外の残酷な真実を映し出す**「構造的アイロニー」**。
    *   **特徴**: 感情語を徹底的に排除し、周囲の風景・物のディテール（雨の匂い、時計の音、光の角度、物の配置）に感情を仮託する**客観的相関物（Objective Correlative）**の手法を用いる。読者は語られなかった「空白」を想像し、冷徹で不気味な余韻を味わう。
    *   **視点**: 制限された一人称、またはシニカルな観察者。「読者が解けるように書く」フェアネスを重視。
    *   **用途**: 診断シーン前半、謎解き、苦い結末（Bitter Aftertaste）。

3.  **Tapestry Mode (社会・多声モード)**
    *   **スタイル**: 宮部みゆき的アプローチ。（参考: 『火車』『理由』— クレジットカード破産・家族崩壊等の社会問題）
    *   **核心原理**: 登場人物の感情を、その置かれた社会的・経済的環境（貧困・格差・借金苦）に対する自然な反応として描く**「共感のリアリズム」**。複数の視点を重ね合わせることで、単一の視点では見えない事件の全体像を**モザイク画**のように浮かび上がらせる。
    *   **特徴**: 加害者・被害者・家族・隣人など無数の関係者の視点を切り替える**パノラマ的視点**。日々の食事・家計のやりくり・湯呑みの位置・掃除道具といった**「生活のディテール」**を丹念に積み上げ、「自分もこうなるかもしれない」という切実なリアリティを生む。抽象的な社会批判は禁止。
    *   **視点**: 多声的（Polyphonic）。主役以外の「声」や、噂、報道などの視点をシームレスに織り交ぜるゼロ焦点化。
    *   **用途**: 世界観の描写、複数のエピソードの接続、群像劇パート。

## Commands

### 1. `generate-character`

算命学（Sanmei）のデータに基づき、3つのモードのいずれか（または融合）を反映したキャラクター設定を生成する。

**Usage**:
これまでのキャラクター生成プロンプトに加え、以下の「モード別・内面の葛藤」を付与して生成せよ。

*   **Somatic**: 「自意識過剰」と「承認欲求」の具体的な現れ方。身体に出る癖。
*   **Logic**: その人物が世界から身を守るために使う「論理の盾（または独自のルール）」。
*   **Tapestry**: その人物を縛る「社会的・経済的な鎖（具体的な生活苦やしがらみ）」。

### 2. `construct-plot`

エピソードの構成を構造化する。

**Usage**:
指定された構造モデルに従ってプロットを出力せよ。

*   **Hub-and-Spoke (感情のハブ)**: [Somatic向け] バラバラの過去のエピソードが、一人の人物（または一つの「感情的真実」）に収束する。客観的事実の整合性よりも、感情の収束点（Emotional Truth）を重視して構成せよ。
*   **Domino Chain (論理の連鎖)**: [Logic向け] 小さな「日常の謎」の解決が、帰納的論理の積み重ねにより必然的に次の大きな謎を呼び、最後に予想外の残酷な真実を映し出す（構造的アイロニー）。フェアネス原則を遵守し、推理に必要な情報はすべてテキスト内に提示せよ。
*   **Ripple Effect (波紋の拡大)**: [Tapestry向け] 一つの事件が、無関係に見えた人々の生活に波及し、複数の視点のモザイク画を通じて最終的に巨大な社会構造を浮かび上がらせる。各視点人物に固有の「生活のディテール」を付与せよ。

### 3. `write-scene`

具体的な本文（シーン）を執筆する。

**Usage**:
`Please write a scene for [Episode/Character] using [Mode].`

**Mode-Specific Instructions (AIへの指示)**:

*   **If Mode is Somatic**:
    > 禁止事項: 「悲しい」「嬉しい」「怒り」「辛い」などの感情ラベルを直接使うこと。
    > 強制事項:
    > - 感情はすべて「呼吸」「体温」「痛み」「五感の歪み」で表現せよ（ソマティック・マーカー）。
    > - 主人公の思い込みや偏見を「事実」として記述せよ（地の文を歪めよ = 信頼できない語り手）。
    > - 嫉妬・自己嫌悪・承認欲求など「直視したくない感情」を容赦なく高解像度で言語化せよ。
    > - 客観的事実よりも「感情的真実（Emotional Truth）」— 登場人物同士が痛みを理解し合えるか — を優先せよ。

*   **If Mode is Logic**:
    > 禁止事項: 安易な共感、感傷的な独白、ご都合主義的な解決（Deus ex machina）。
    > 強制事項:
    > - 事実関係を淡々と記述せよ。環境音、光の角度、物の配置などの「ノイズ」を精密に描写し、そこから登場人物の押し殺した感情を読者に「推測」させよ（客観的相関物）。
    > - 帰納的論理を用いよ：小さな謎から出発し、論理を積み重ね、その完成図が予想外の残酷な真実を映し出す構造的アイロニーを構築せよ。
    > - 「読者が解けるように書く」フェアネスを守れ：推理に必要な情報はすべて提示すること。
    > - 結末にはビターな余韻（Bitter Aftertaste）— 完全には割り切れない苦味 — を残せ。

*   **If Mode is Tapestry**:
    > 禁止事項: 抽象的な社会批判。単一の視点への固執。
    > 強制事項:
    > - 「生活の手触り」を書け。財布の中身、夕食の献立、職場の人間関係の摩擦、湯呑みの位置、掃除道具など、具体的な生活ディテールから社会的な重荷を描写せよ。
    > - パノラマ的視点を用いよ：加害者・被害者・家族・隣人など複数の関係者の視点を切り替え、モザイク画のように事件の全体像を浮かび上がらせよ。
    > - 共感のリアリズム：登場人物の感情を、社会的・経済的環境に対する自然な反応として描け。読者に「自分もこうなるかもしれない」と感じさせよ。

### 4. `auto-mode-select`

シーンの内容に応じて最適なモードを提案する。

*   **診断パート（問診〜分析）**: Logic Mode (論理的分析)
*   **回想・告白パート**: Somatic Mode (感情の吐露)
*   **解決・処方箋パート**: Hybrid (Logicで構造を作り、Somaticで救済する)
