# Novel Critique Skill

このスキルは、AIが生成したテキストが「日本的感性」と「構造的論理」の高い基準を満たしているかを厳格にチェックします。

## Criteria by Mode

### 1. Somatic Critique (身体的感性チェック)
*   **Show, Don't Tell**: 「悲しい」「辛い」「嬉しい」「怒り」などの感情ラベルが使われていないか？（NGワード検出）
*   **Somatic Marker**: 感情が「喉の詰まり」「指先の冷え」「胃の痛み」「呼吸が浅くなる」「視界が歪む」などの身体反応として克明に記述されているか？ 解像度は十分か？
*   **Bias（信頼できない語り手）**: 語り手の主観的な思い込みや、感情バイアスによる事実の歪曲が地の文に織り込まれているか？ 主人公の偏見が「事実」として記述されているか？
*   **Emotional Truth**: 客観的事実の整合性だけでなく、「感情的真実」— 登場人物同士が痛みを理解し合えるか — が物語の収束点になっているか？
*   **Dark Emotions**: 嫉妬・自己嫌悪・承認欲求など「直視したくない感情」を回避せず、容赦なく言語化しているか？

### 2. Logic Critique (論理・抑制チェック)
*   **Negative Space**: 登場人物が感情を押し殺し、沈黙や「語らないこと」で表現しているか？ 感情が過剰に漏れ出ていないか？
*   **Objective Correlative（客観的相関物）**: 感情が風景（雨、無機質な部屋、時計の音）、光の角度、物の配置などの環境描写に仮託されているか？
*   **Inductive Logic（帰納的論理）**: 小さな謎から出発し、論理の積み重ねで真相に到達する構造になっているか？ 論理的飛躍がないか？
*   **Structural Irony（構造的アイロニー）**: 論理の完成図が予想外の残酷な真実を映し出す仕掛けがあるか？
*   **Fairness（フェアネス）**: 推理に必要な情報がすべてテキスト内に提示されているか？ 「読者が解ける」構造になっているか？
*   **Bitter Aftertaste**: 結末に皮肉や、完全には割り切れない「余韻（苦味）」があるか？ 安易なハッピーエンドに逃げていないか？

### 3. Tapestry Critique (社会・多声チェック)
*   **Life Details（生活のディテール）**: 金額、食事の内容、居住環境、湯呑みの位置、掃除道具など、具体的で触れるほどの生活描写があるか？ 抽象的な社会批判に逃げていないか？
*   **Social Connection（共感のリアリズム）**: 個人の悩みが社会構造（制度、経済状況、貧困、格差、借金苦）と接続されているか？ 登場人物の感情が社会的環境の自然な反応として描かれているか？ 読者に「自分もこうなるかもしれない」と感じさせる切実さがあるか？
*   **Polyphony（多声性）**: 主人公以外の「声」（加害者、被害者、家族、隣人、噂、報道）が適切に織り込まれているか？ 複数の視点のモザイク画で事件の全体像が浮かび上がるか？
*   **Panoramic Perspective（パノラマ的視点）**: 単一の視点に固執せず、視点の切り替えが円滑か？ ゼロ焦点化が効果的に機能しているか？

### 4. Realism & Consistency Critique (リアリティ・一貫性チェック)
*   **Cross-Episode Character Isolation**: 他のエピソードの登場人物が混入していないか？ 各エピソードの登場人物は独立しており、別エピソードの固有名（森川陽菜など）が紛れ込んでいないか？
*   **Dialogue Tone（台詞のトーン検証）**: 巡の台詞が「押しつけ」「断定」になっていないか？ 巡は診断結果を「そうでしょう」と決めつけるのではなく、「〜だったのか」「〜なのかもしれない」と問いかけ・独り言のように語るのが原則。相手の内面を勝手に代弁していないか？
*   **Diagnostic Flow（診断フローの完備）**: 初回来訪者に対して、①名前・生年月日の取得、②サービス説明（何をする場所か）、③料金・所要時間の提示、④同意の確認——が漏れなく存在するか？
*   **Temporal Consistency（時間整合性）**: 昼休みに訪問→診断時間→帰還の時間経過が現実的か？ 昼が夕方にならないか？
*   **Information Origin（情報出所の整合）**: 巡が「高橋さん」と呼ぶなら、どこで名前を知ったか（カード記入など）が明示されているか？ 根拠なく登場人物の名前・情報を使用していないか？
*   **Number Rule（数字の発話禁止）**: 五行の数値（木128、火0など）が巡の内面描写のみに留まり、台詞として口に出していないか？

## Commands

### 1. `review-sensibility`

Somatic Modeの基準でチェックを行う。

**Usage**:
`Input text` を分析し、直接的な感情語（NGワード）をリストアップし、身体的描写への書き換え案を提示せよ。

### 2. `review-logic`

Logic Modeの基準でチェックを行う。

**Usage**:
`Input text` の論理構造を分析し、ご都合主義な展開（Deus ex machina）や感情過多な表現を指摘せよ。

### 3. `review-social`

Tapestry Modeの基準でチェックを行う。

**Usage**:
`Input text` に含まれる「生活の臭い」の不足を指摘し、追加すべき社会的背景（職業的リアリティなど）を提案せよ。

### 4. `review-all`

総合的な批評を行う。
